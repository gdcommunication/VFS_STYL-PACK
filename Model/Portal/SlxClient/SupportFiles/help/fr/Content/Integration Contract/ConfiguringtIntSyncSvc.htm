Configuration du service de synchronisation d'intégration

Le service de synchronisation d'intégration (service de synchronisation SDataSaleslogix) permet le transfert des données entre Saleslogix et vos intégrations. Le service doit être démarré et configuré pour s'exécuter de manière automatique.

La synchronisation des informations en temps réel requiert une connexion Internet. Lors de l'utilisation de l'intégration du système de comptabilité pour le Client Web hors ligne, les vues des données en temps réel sont actualisées à la dernière connexion du client à la base de données hôte.

Pour configurer

  1. Connectez-vous comme utilisateur WebDLL.
  2. Veillez à configurer et à activer la Synchronization SData et votre ou vos intégration(s) avant de démarrer le service.
  3. Consultez le fichier Sage.SalesLogix.SLXSDataSyncServer.exe.config pour vérifier que la connexion est définie correctement à la fin du fichier. Les informations de connexion doivent correspondre au fichier connection.config déployé sur votre site Web.
    Par défaut, le fichier de configuration est ...\Program Files\Saleslogix\Sage.SalesLogix.SLXSDataSyncServer.exe.config et le fichier de connexion est déployé vers ...\inetpub\wwwroot\slxclient
  4. Vérifiez que l'authentification Windows a été activée pour l'utilisateur WebDLL.
  5. Sur le poste de travail administratif, affichez Services Windows sous Gestion de l'ordinateur.
  6. Cliquez le bouton droit de la souris sur Saleslogix Serveur de synchronisation SData, puis cliquez sur Propriétés.
    1. Sélectionnez l'option Automatique dans la zone Type de démarrage.
    2. Configurez le service pour se connecter et s'exécuter en tant qu'utilisateur WebDLL.
    3. Cliquez sur Démarrer.
  7. Cliquez sur OK.
