Modification des champs de texte SQL

L'éditeur des champs de publipostage vous permet de modifier les champs de fusion de texte SQL, qu'ils soient nouveaux ou non. Lorsque vous sélectionnez un champ de texte SQL dans la vue Liste, les options de modification sont affichées dans le volet de modification.

  • Identificateur de champ : permet de modifier le texte affiché dans le modèle.
  • SQL : permet d'entrer une instruction SELECT pour demander des données au fournisseur OLE DB. Seul le premier enregistrement de la première colonne est affiché.
    Cliquez sur le bouton Modifier pour ouvrir la boîte de dialogue Modifier SQL.
  • Formater comme : affiche le type du champ de fusion.
  • Type de format : vous permet de modifier le type de formatage appliqué au champ de fusion
  • sélectionné.
  • Chaîne de formatage : permet de formater la valeur affichée dans le champ de fusion. Avant d'ajouter une chaîne de formatage, vous devez définir le type de format. Les formats de chaîne valides dépendent du type de format que vous sélectionnez.
  • Texte avant : permet d'entrer le texte affiché avant le champ de fusion sélectionné.
  • Texte après : permet d'entrer le texte affiché après le champ de fusion sélectionné.
  • Format spécial : permet d'appliquer un formatage de caractère spécial au champ de fusion (y compris au texte affiché avant et après le champ de fusion).
  • Hyperlink Address (Adresse du lien hypertexte) : permet d'entrer l'URL à ouvrir à partir du champ de fusion.
     Exemples d'adresse de lien hypertexte
    • http://www.saleslogix.com/
    • mailto:jabbott@abbott.demo?Subject=Special Offer&Body=Yes, please send me info on this special offer!
    • mailto::SLXUserName <:SLXUserEmail>?Subject=Special Offer&Body=Yes, please send me info on this special offer!
    • ftp://ftp.winzip.com/winzip81.exe
    • news://msnews.microsoft.com/microsoft.public.word.mailmerge.fields
  • Hyperlink Target (Cible du lien hypertexte) : permet de choisir le mode d'ouverture de l'URL. Dans la liste déroulante, sélectionnez une cible.
    • Option cible Description
    Paramètre par défaut de page (aucun) Ouvre la cible du lien dans la même fenêtre.
    Même cadre

    Ouvre la cible du lien dans le même cadre.

    Correspond à la cible HTML « _self ».
    Page entière

    Ouvre la cible du lien dans la totalité de l'espace de la fenêtre actuelle.

    Correspond à la cible HTML « _top ».
    Nouvelle fenêtre

    Ouvre la cible du lien dans une nouvelle fenêtre sans titre.

    Correspond à la cible HTML « _blank ».
    Cadre parent

    Ouvre la cible du lien dans la fenêtre immédiatement parente du document contenant le lien.

    Correspond à la cible HTML « _parent ».
  • Info-bulle du lien hypertexte - vous permet de saisir du texte qui apparaît lorsque le pointeur de la souris est placé sur le lien hypertexte.
     Des paramètres peuvent être inclus dans les champs Adresse et Info-bulle du lien hypertexte ; ils peuvent s'avérer pratiques lors de la création d'un lien hypertexte contenant une adresse (mailto:).
    • Saleslogix Paramètre Description
    SLXAccountManagerEmail Le paramètre :SLXAccountManagerEmail sera remplacé par l'adresse électronique du responsable du compte de l'enregistrement du contact fusionné.
    SLXAccountManagerName Le paramètre :SLXAccountManagerName sera remplacé par le nom du responsable du compte de l'enregistrement du contact fusionné.
    SLXAccountManagerTitle Le paramètre :SLXAccountManagerTitle sera remplacé par la fonction du responsable du compte de l'enregistrement du contact fusionné.
    SLXAccountManagerPhone Le paramètre :SLXAccountManagerPhone sera remplacé par le numéro de téléphone du responsable du compte de l'enregistrement du contact fusionné.
    SLXAccountManagerFax Le paramètre :SLXAccountManagerFax sera remplacé par le numéro de fax du responsable du compte de l'enregistrement du contact fusionné.
    SLXUserEmail Le paramètre :SLXUserEmail sera remplacé par l'adresse électronique de l'utilisateur du publipostage.
    SLXUserName Le paramètre :SLXUserName sera remplacé par le nom de l'utilisateur du publipostage.
    SLXUserTitle Le paramètre :SLXUserTitle sera remplacé par la fonction de l'utilisateur du publipostage.
    SLXUserPhone Le paramètre :SLXUserPhone sera remplacé par le numéro de téléphone de l'utilisateur du publipostage.
    SLXUserFax Le paramètre :SLXUserFax sera remplacé par le numéro de fax de l'utilisateur du publipostage.
