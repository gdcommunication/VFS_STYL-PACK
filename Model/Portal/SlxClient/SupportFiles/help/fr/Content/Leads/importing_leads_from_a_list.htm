Importation de leads à partir d'une liste

Utilisez la vue Importer des leads pour importer des listes de leads dans Saleslogix. Vous pouvez uniquement importer des leads d'une liste CSV (valeurs séparées par des virgules). Les types de fichiers valides comprennent les fichiers Microsoft Excel, .txt et d'autres types de fichier de base de données mais les fichiers doivent être enregistrés au format CSV.

Les fichiers texte doivent être de type ANSI ou ASCII.

Pour importer des leads

  1. Dans le menu Outils, cliquez sur Importer des leads.
  2. Remplissez les informations de Sélection d'un fichier puis cliquez sur Suivant.
  3. Remplissez les informations de Définition d'un séparateur puis cliquez sur Suivant.
  4. Remplissez les informations de Mappage de champs puis cliquez sur Suivant.
  1. Remplissez les informations de Gestion des doublons et cliquez sur Suivant.
  2. Remplissez les informations de Sélection d'actions d'importation, puis cliquez sur Suivant.
  3. Passez en revue les sélections d'importation et cliquez sur Envoyer.
  4. Contrôlez la demande de processus d'importation.
  5. Quand l'importation est terminée, cliquez sur le lien Numéro d'importation pour afficher les résultats de l'historique des importations dans la vue Historique des importations.
