Modification de l'état d'un processus de contact

Le client Web vous permet de suspendre ou d'abandonner un processus et de relancer un processus suspendu. Les processus abandonnés sont définitivement annulés.

Les utilisateurs WebViewer ne pourront pas accéder à cette fonction.

Pour abandonner ou suspendre un processus

  1. Dans la barre de navigation, cliquez sur Processus.
  2. Dans la vue Processus, cliquez sur un onglet pour afficher tous les processus ayant un état spécifique. Seuls les processus dont l'état est Lancé, En cours, En attente ou Suspendu peuvent être abandonnés.

    		3.

    Vous ne pouvez pas modifier l'état d'un processus de contact à partir des onglets Tous les Processus ou Terminé.
  3. Pour suspendre le processus, cliquez sur le point jaune dans la colonne Modifier l'état.
  4. Pour abandonner le processus, cliquez sur le point rouge dans la colonne Modifier l'état.

Pour relancer un processus suspendu

  1. Dans la barre de navigation, cliquez sur Processus.
  2. Dans la vue Gestionnaire de processus, cliquez sur l'onglet Suspendu pour afficher tous les processus ayant un état spécifique.
  3. Dans la colonne Changer, cliquez sur le point vert.
