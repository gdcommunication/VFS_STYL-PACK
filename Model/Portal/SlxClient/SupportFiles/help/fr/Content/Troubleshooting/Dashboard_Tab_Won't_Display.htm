Un onglet de mon Tableau de bord ne s'affiche ou ne s'ouvre pas sur un autre site Web

Problème :

Quand j'essaie d'afficher un onglet sur mon Tableau de bord, un des scénarios suivants se produit :

  • L'onglet ne s'ouvre pas.
  • Je suis immédiatement conduit à un autre site web.

Solution :

L'onglet a un widget de site web qui accède à un site web non optimisé pour l'utilisation d'un widget. Vous devez supprimer l'onglet, puis le recréer.

Pour éviter cela à l'avenir, vous devez tout d'abord créer un onglet « test » et y ajouter le widget du site web pour vous assurer que le site web se comporte comme prévu avant d'ajouter un widget de site web à un onglet existant.
