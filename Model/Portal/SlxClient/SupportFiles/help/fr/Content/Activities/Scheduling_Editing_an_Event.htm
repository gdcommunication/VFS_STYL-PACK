Planification ou modification d'un événement

Les événements peuvent être planifiés, modifiés, supprimés et consultés dans le client Web.

Si vous utilisez Intellisync for Saleslogix pour synchroniser des activités et des événements avec Microsoft Outlook , les doublons seront générés dans les cas suivants :

  • si vous créez l'événement dans Saleslogix, et
  • synchronisez l'événement sur MS Outlook puis
  • dans MS Outlook transformez l'événement en événement périodique.

Lorsque l'événement sera synchronisé dans Saleslogix, il apparaîtra comme une activité périodique sans horaire défini, mais l'événement d'origine figurera aussi sur l'agenda. Pour éviter toute confusion, vous pouvez supprimer manuellement l'événement d'origine. Reportez-vous à la rubrique Suppression d'un événement pour plus d'informations.

Pour planifier un événement

  1. Dans le menu Planifier, cliquez sur Evénement.
  2. Dans la zone Date de début, cliquez sur Bouton  Agenda   pour sélectionner la date à laquelle l'événement aura lieu, puis cliquez sur OK.

Si vous n'utilisez pas la date de début par défaut, une fois la nouvelle date de début saisie, la date de fin est automatiquement mise à jour afin de correspondre à la date de début.

  1. Dans la zone Date de fin, cliquez sur Bouton  Agenda   pour sélectionner la date à laquelle l'événement aura lieu, puis cliquez sur OK.
  2. Dans la zone Type événem., Cliquez sur    et sélectionnez le type d'événement.
  3. Dans la zone Emplacement, saisissez un emplacement pour l'événement.
  4. Dans la zone Description, entrez les informations sur l'événement.
  5. Dans la zone Utilisateur, cliquez sur le Bouton  Rechercher   et utilisez la recherche pour sélectionner l'utilisateur pour lequel l'événement est planifié.

  6. Cliquez sur OK.

Pour afficher ou modifier un événement de la vue Activités

  1. Ouvrez la vue Liste des activités, puis cliquez sur Evénements.
  2.  Si nécessaire, utilisez les filtres pour restreindre la liste.
  3. Cliquez sur la date de début d'un événement pour l'ouvrir. 
  4. Si nécessaire, modifiez les informations de l'événement.

  5. Cliquez sur OK.
