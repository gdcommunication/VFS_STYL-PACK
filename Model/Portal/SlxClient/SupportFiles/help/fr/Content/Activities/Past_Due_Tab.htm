Onglet Activités échues

L'onglet Activités échues dans la vue Activités affiche toutes vos activités échues. Une activité est considérée comme échue lorsque la date de début programmée a expiré et que l'activité n'a pas d'alarme.

Procédure

Afficher vos activités échues
  1. Dans la vue Activités, cliquez sur Activités échues ou dans la barre d'outils, cliquez sur le bouton Vous avez des activités échues.
  2. Si nécessaire, utilisez les filtres pour restreindre la liste.
  3. Cliquez sur le type de l'activité pour ouvrir l'activité. 
Afficher ou masquer le volet des détails
  1. Cliquez sur Détails.
  2. Sélectionnez un enregistrement.
    Le volet des détails apparaît en bas de la vue Liste et affiche des informations détaillées sur l'enregistrement sélectionné.
  1. Pour masquer le volet des détails, cliquez sur Masquer les détails.
Afficher le volet de résumé
  1. Cliquez sur Résumé.
  2. Pour retourner vers la vue Liste, cliquez sur Liste.

Utiliser le menu contextuel

Terminer une activité
  1. Effectuez l'une des opérations suivantes :
    • Dans la vue Résumé ou sur le volet des détails, cliquez sur le bouton Terminer de l'activité appropriée.
    • Sélectionnez l'activité à terminer et dans le Volet des tâches, dans la zone Tâches courantes, cliquez sur Terminer l'activité.
    • Cliquez avec le bouton droit de la souris sur l'activité à terminer, puis sélectionnez Terminer l'activité.
  2. Suivez les étapes pour terminer une activité.

Terminer plusieurs activités

Supprimer une activité
  1. . Sélectionnez l'activité à supprimer et dans le Volet des tâches, dans la zone Tâches courantes, cliquez sur Supprimer l'activité.
  2.  Cliquez sur OK.
Supprimer plusieurs activités
  1. Sélectionnez les activités à supprimer.
  2. . Dans le Volet des tâches, dans la zone Tâches courantes, cliquez sur Supprimer l'activité.
  3.  Cliquez sur OK.
Planifier un appel téléphonique

  1. Effectuez l'une des opérations suivantes :

    • Dans le Volet des tâches Tâches courantes, sous Planifier, cliquez sur Appel téléphonique.
    • Cliquez avec le bouton droit dans la grille, pointez sur Planifier et cliquez sur Appel téléphonique.
  2. Suivez les étapes de planification d'un appel téléphonique.
Planifier une réunion

  1. Effectuez l'une des opérations suivantes :

    • Dans le Volet des tâches Tâches courantes, sous Planifier, cliquez sur Réunion.
    • Cliquez avec le bouton droit de la souris dans la grille, pointez sur Planifier, puis cliquez sur Réunion.
  2. Suivez les étapes de planification d'une réunion.
Planifier une tâche à faire

  1. Effectuez l'une des opérations suivantes :

    • Dans le Volet des tâches Tâches courantes, sous Planifier, cliquez sur Tâche à faire.
    • Cliquez avec le bouton droit de la souris dans la grille, pointez sur Planifier, puis cliquez sur Tâche à faire.
  2. Suivez les étapes de planification d'une tâche à faire.
Planifier une activité personnelle

  1. Effectuez l'une des opérations suivantes :

    • Dans le Volet des tâches Tâches courantes, sous Planifier, cliquez sur Activité personnelle.
    • Cliquez avec le bouton droit de la souris dans la grille, pointez sur Planifier, puis cliquez sur Activité personnelle.
  2. Suivez les étapes de planification d'une activité personnelle.
