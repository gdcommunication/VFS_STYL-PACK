Zones d'informations sur le compte

Les zones d'informations sur le compte contiennent des informations sur le compte sélectionné. Vous pouvez si nécessaire modifier ou supprimer ces informations.

Rectangle Description/Action
Compte Société avec laquelle vous entretenez actuellement des relations commerciales ou qui fait partie de vos prospects.
Division Dans la société, la division à laquelle le compte appartient.
Adresse

Adresse ou emplacement du compte. Cliquez sur le bouton Modifier, pour ouvrir la boîte de dialogue Modifier l'adresse.

Pour afficher une carte de l'adresse, cliquez sur le bouton MapQuest.
Principal Numéro de téléphone principal du compte. Cliquez dans la zone et saisissez les modifications éventuelles.
Fax Numéro de fax du compte. Cliquez dans la zone et saisissez les modifications éventuelles.
Numéro vert Numéro vert du compte. Cliquez dans la zone et saisissez les modifications éventuelles.
URL Web Adresse Internet du compte. Cliquez dans la zone et saisissez les éventuelles modifications. Pour ouvrir le site Web du contact, cliquez sur le bouton WWW.
Propriétaire Utilisateur ou équipe d'utilisateurs disposant de droits d'accès au compte. Cliquez sur le bouton Rechercher pour rechercher l'utilisateur ou l'équipe au sein de votre société qui a accès au compte.
Resp. de compte  Personne ou équipe de votre entreprise qui gère les relations avec le compte.Cliquez sur le bouton Rechercher  pour rechercher l'utilisateur ou l'équipe de votre entreprise qui gère les relations avec le compte.
Type Catégorie du compte. Cliquez sur    et sélectionnez un élément dans la liste.
Sous type La sous-catégorie du compte. Cliquez sur    et sélectionnez un élément dans la liste.
Status

Situation actuelle du compte. Cliquez sur    et sélectionnez un élément dans la liste.

La liste peut contenir les éléments Actif, Dormant, Suspens de crédit comme exemple.
Industrie Le secteur du compte. Cliquez sur    et sélectionnez un élément dans la liste.
Desc.  de l'activité Une description de l'activité du compte.
