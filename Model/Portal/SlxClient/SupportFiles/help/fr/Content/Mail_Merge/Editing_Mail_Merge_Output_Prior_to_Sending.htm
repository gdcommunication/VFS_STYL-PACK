Modification de la sortie de publipostage avant l'envoi

Vous pouvez modifier le document fusionné avant de l'envoyer à certains destinataires. Vous pouvez, par exemple, ajouter un mot personnel à une seule lettre. Cette option se trouve dans l'onglet Options de publipostage.

Pour modifier la sortie de publipostage avant l'envoi

  1. Dans le menu Rédiger, sélectionnez Publipostage.
  2. Dans l'onglet Options de fusion, renseignez les zones Modèle, Fusionner avec et Sortie.
  3. Dans la zone Options de fusion, sélectionnez la première option nommée Modifier les différents <documents> avant de les envoyer.
  4. Renseignez les autres zones d'information, si nécessaire puis cliquez sur Fusionner.

Le moteur de fusion affiche la progression et il faudra peut-être sélectionner une adresse de contact.

  1. Dans la boîte de dialogue Modifier les documents fusionnés, sélectionnez le document à modifier, puis cliquez sur Ouvrir.
  2. Dans l'éditeur, effectuez vos modifications ou ajoutez un mot personnel au document.
  3. Sur la barre d'outils, cliquez sur Enregistrer et fermer.
  4. (Facultatif) Sélectionnez un autre document, cliquez sur Ouvrir et suivez les étapes 6 et 7 pour chaque document.
  5. Une fois que toutes les modifications de document sont effectuées, cliquez sur Terminé.

Les documents fusionnés sont alors traités selon votre choix de sortie.
