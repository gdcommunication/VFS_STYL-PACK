Rédaction et envoi d'un e-mail

Vous pouvez rédiger un e-mail pour un seul contact ou lead en utilisant l'option Rédiger un e-mail ou Rédiger un e-mail à partir d'un modèle. Si vous devez envoyer un e-mail à plusieurs contacts ou leads, utilisez Publipostage.

Microsoft Outlook doit être installé sur votre ordinateur pour pouvoir utiliser les fonctionnalités de Rédiger un E-mail.

Si le menu Rédiger n'est pas disponible, vous devez télécharger l'intégration du bureau Saleslogix. Vous pouvez installer l'intégration du bureau Saleslogix depuis l'écran de connexion du client Web. 
Pour écrire un e-mail sans utiliser de modèle
  1. Effectuez l'une des opérations suivantes :
    • Dans le menu Rédiger, pointez sur E-mail.
    • Cliquez sur le bouton Envoyer par e-mail en regard de l'adresse électronique du contact ou du lead.
  2. Si la messagerie est activée, mais que vous n'êtes pas connecté, cliquez sur Oui dans la boîte de messages pour vous connecter.

  3. Your next step depends on your focus within the Client Web Saleslogix.

    • If your focus is on a particular contact or lead, the merge will continue.
    • If you do not have focus on a particular contact or lead, use the Select Contact or Select Lead dialog box to select a contact or lead.
    • If your focus is on an account or an opportunity, the Select Contact dialog box will prompt you to select a contact.
    • If your focus is on a ticket the document will merge with the associated contact.

  4. Complétez la zone Objet et le corps du message de la fenêtre du nouvel e-mail.
  5. (Facultatif) Si le message électronique requiert une pièce jointe, cliquez sur Insérer, puis cliquez sur une des options de menu. Si vous ajoutez un fichier, consultez A propos des pièces jointes d'e-mail pour plus de détails.
  6. Send your e-mail by doing one of the following:
    • Cliquez sur Envoyer SLX. Si vous avez joint un fichier et souhaitez l'enregistrer en tant que copie, cliquez sur Oui lorsque vous y êtes invité. L'élément d'e-mail est enregistré dans l'historique du contact et le fichier est affiché dans l'onglet Pièces jointes.
    • Cliquez sur Envoyer. Le message n'est pas automatiquement enregistré dans l'historique du contact ; cependant, vous pouvez faire glisser et déplacer le message depuis MS Outlook dans Saleslogix Saleslogix.

      		•

      Les fichiers graphiques plus petits que la taille minimum définie par votre administrateur ne seront pas enregistrés comme pièces jointes séparées.

  7. Utilisez la boîte de dialogue Terminer une activité pour modifier les informations de l'historique, saisir des notes ou planifier une activité de suivi, puis cliquez sur Terminer. 

  8. Si vous avez choisi de planifier une activité de suivi, complétez la boîte de dialogue du type d'activité.
Pour rédiger un e-mail en utilisant un modèle
  1. Dans le menu Rédiger, pointez sur E-mail à partir d'un modèle et cliquez sur un nom de modèle ou sur Autres modèles pour sélectionner un autre modèle. Si vous êtes invité à ajouter le modèle à la liste des modèles les plus récemment utilisés, cliquez sur Oui pour ajouter le modèle à la liste ou bien cliquez sur Non.
  2. Si la messagerie est activée, mais que vous n'êtes pas connecté, cliquez sur Oui dans la boîte de messages pour vous connecter.

  3. Your next step depends on your focus within the Client Web Saleslogix.

    • If your focus is on a particular contact or lead, the merge will continue.
    • If you do not have focus on a particular contact or lead, use the Select Contact or Select Lead dialog box to select a contact or lead.
    • If your focus is on an account or an opportunity, the Select Contact dialog box will prompt you to select a contact.
    • If your focus is on a ticket the document will merge with the associated contact.

  4. Si vous avez sélectionné Autres modèles à l'étape 1, dans la boîte de dialogue Gestion des modèles, sélectionnez un modèle et cliquez sur OK. Si vous y êtes invité, choisissez une adresse dans la boîte de dialogue Sélectionner le type d'adresse.
    L'adresse électronique du contact est affichée dans la zone A ; la zone Objet contient les informations enregistrées dans le modèle et une identification des pièces jointes au modèle est affichée.
  5. (Facultatif) Dans la zone de texte, vous pouvez taper des informations supplémentaires ou changer la signature par défaut.
  6. (Facultatif) Si le message électronique requiert une pièce jointe, cliquez sur Insérer, puis cliquez sur une des options de menu. Si vous ajoutez un fichier, consultez A propos des pièces jointes d'e-mail pour plus de détails.
  7. Send your e-mail by doing one of the following:
    • Cliquez sur Envoyer SLX. Si vous avez joint un fichier et souhaitez l'enregistrer en tant que copie, cliquez sur Oui lorsque vous y êtes invité. L'élément d'e-mail est enregistré dans l'historique du contact et le fichier est affiché dans l'onglet Pièces jointes.
    • Cliquez sur Envoyer. Le message n'est pas automatiquement enregistré dans l'historique du contact ; cependant, vous pouvez faire glisser et déplacer le message depuis MS Outlook dans Saleslogix Saleslogix.

      		•

      Les fichiers graphiques plus petits que la taille minimum définie par votre administrateur ne seront pas enregistrés comme pièces jointes séparées.

  8. Utilisez la boîte de dialogue Terminer une activité pour modifier les informations de l'historique, saisir des notes ou planifier une activité de suivi, puis cliquez sur Terminer. 

  9. Si vous avez choisi de planifier une activité de suivi, complétez la boîte de dialogue du type d'activité.
