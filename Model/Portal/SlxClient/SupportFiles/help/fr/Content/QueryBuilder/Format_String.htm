Chaîne de formatage

Utilisez une chaîne de formatage pour mettre en forme les données affichées dans la requête. Sélectionnez tout d'abord Type de format. Saisissez ensuite une chaîne de formatage pour modifier le type sélectionné. Les options valides dépendent du type de format sélectionné. Une chaîne de formatage doit commencer par un signe % et se terminer par un caractère de type de format :

[%] [type]

Le cas échéant, vous pouvez aussi inclure :

  • un spécificateur d'index d'arguments
  • un indicateur de justification à gauche
  • un spécificateur de largeur
  • un spécificateur de précision

[%] [index:] [-] [width] [.precision] [type]

Pour indiquer le nombre de décimales à employer dans une chaîne de formatage, définissez la chaîne au format correspondant. Ajoutez ensuite un spécificateur de précision, soit une virgule et le nombre de chiffres après cette virgule, avant le caractère de chaîne de formatage mais après le % de début. Exemple :

  • Type de champ : Float
  • Type de format : ftCurrency.
  • Chaîne de formatage : %.4f
  • Données (non formatées) : 3.1415926&ldots;
  • Résultat : 3.1416

Vous pouvez utiliser les types de chaînes de format suivants à votre convenance :

  • Chaînes de format de champ booléen
  • Chaînes de format de champ Devise et Téléphone
  • Chaînes de format de champ Date/heure
  • Chaînes de format de champ fixe
  • Chaînes de format de champ Entier
