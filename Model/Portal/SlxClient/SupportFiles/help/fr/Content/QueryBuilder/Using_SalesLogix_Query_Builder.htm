Présentation de Query Builder  

Query Builder vous permet de sélectionner des données dans la base de données en utilisant une ou plusieurs conditions. Vous pourriez par exemple avoir besoin de consulter la liste des contacts principaux dans le secteur bancaire. Query Builder est connecté à différents domaines dans tous les clients Saleslogix. Pour utiliser Query Builder de façon efficace, vous devez comprendre la structure de la base de données Saleslogix et les concepts de bases de données relationnelles, comme par exemple les relations entre les données et les jointures.
