Exportation d’un groupe vers MS Excel

Vous pouvez exporter des groupes vers MS Excel. Les en-têtes de colonnes et la mise en page des groupes seront aussi exportées.

Si l'option Exporter vers MS Excel n'est pas disponible, vous devez télécharger l'intégration du bureau Saleslogix. Vous pouvez installer l'intégration du bureau Saleslogix depuis l'écran de connexion du client Web. 

Pour exporter un groupe vers un fichier

  1. Affichez le groupe ou les résultats de la recherche contenant les enregistrements que vous voulez exporter.
  2. Sur une vue Liste, cliquez avec le bouton droit sur l'onglet du groupe et cliquez sur Exporter vers MS Excel.
    Microsoft Excel s'ouvre avec le groupe exporté. Si le groupe contient des informations de devises, MS Excel crée également un tableau croisé. MS Excel ajoute aussi une ligne de totaux à chacune des colonnes contenant des informations de devises. Vous pouvez ainsi afficher le total du groupe.
  • L'exportation d'un groupe vers MS Excel entraîne l'exportation du groupe entier, quels que soient les filtres que vous avez appliqués.
  • Il se peut que vous receviez un message d'erreur si vous essayez d'exporter un groupe qui inclut un champ de données comportant un grand volume de données.
