Copie d'un groupe

Vous pouvez sélectionner un groupe et le copier. Vous avez la possibilité de conserver les conditions de ce groupe ou de les modifier. Il est impossible de copier des groupes ad hoc.

Pour copier un groupe

  1. Dans une vue Liste, cliquez avec le bouton droit sur l'onglet d'un groupe.
  2. Cliquez sur Copier.  
    Query Builder s'ouvre.
  3. Utilisez les onglets de Query Builder pour modifier le groupe. Onglet
    • Propriétés : utilisez cet onglet pour définir le nom et la description du groupe. Onglet
    • Conditions : utilisez cet onglet pour définir la ou les instruction(s) conditionnelles utilisées pour sélectionner les données. Cet onglet n'est pas disponible pour les groupes ad hoc. Onglet
    • Présentation : utilisez cet onglet pour définir les données qui s'affichent dans la vue Liste.

      La présentation par défaut est définie d'après le groupe que vous avez sélectionné comme groupe par défaut dans vos options de groupe. Onglet

    • Tri : utilisez cet onglet pour définir le mode de tri des données. Onglet
  4. Une fois toutes les modifications effectuées, cliquez sur OK.
