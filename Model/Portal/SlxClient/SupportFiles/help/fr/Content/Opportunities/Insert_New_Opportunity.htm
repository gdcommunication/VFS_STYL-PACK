Ajout d'une nouvelle opportunité

Utilisez la vue Insérer une opportunité pour saisir des informations d'opportunité et pour ajouter, modifier ou supprimer des produits, des concurrents ou des contacts. Vous pouvez également mettre à jour la devise de l'opportunité et son taux si la fonction de devises multiples est activée.

Pour ajouter une nouvelle opportunité

  1. Dans le menu Nouveau, cliquez sur Opportunité.
  2. Compte : cliquez sur le bouton Rechercher pour rechercher le compte à associer à l'opportunité.
  3. Saisissez les informations dans les zones d'informations relatives aux opportunités. 
    Selon vos Options définies pour les opportunités, certaines zones peuvent être déjà renseignées.
    • Description : saisissez un nom pour l'opportunité.
      Pour un meilleur suivi, utilisez le nom de compte défini dans la description de l'opportunité.
    • Compte - Cliquez sur le bouton Rechercher pour rechercher le compte à associer à l'opportunité.
    • Resp. cpte. - Cliquez sur le bouton Rechercher pour rechercher la personne ou l'équipe de votre entreprise qui gère le compte.
    • Revendeur : cCliquez sur le bouton Rechercher pour sélectionner un revendeur associé à cette opportunité.
    • Approx. Clôture - cliquez sur Bouton  Agenda  , sélectionnez la date approximative de clôture de l'opportunité, et cliquez sur OK.
    • Processus de vente : sélectionnez un processus de vente approuvé à utiliser pour cette opportunité.

      La zone Processus de vente ne sera pas disponible si le bundle Saleslogix Advanced n'est pas installé.

    • État - votre société définit la liste des états des opportunités.. Cliquez sur    et sélectionnez un état dans la liste.
    • Type : Cliquez sur   , et sélectionnez dans la liste un type qui décrit l'opportunité.
    • Source du lead - Cliquez sur le bouton Rechercher, Source du leadSource : , pour sélectionner la manière dont l'opportunité a été générée .
    • Probabilité de clôture  : Cliquez sur   , puis sélectionnez la probabilité de clôture de cette opportunité dans la liste.
  4. Ajoutez des produits, des concurrents ou des contacts.
  5. Cliquez sur Enregistrer pour enregistrer l'opportunité et fermer la vue, ou cliquez sur Enregistrer et Nouveau pour enregistrer l'opportunité en cours, et ouvrir une nouvelle opportunité.
