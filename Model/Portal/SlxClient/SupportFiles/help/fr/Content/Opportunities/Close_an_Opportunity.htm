Clôture d'une opportunité

Lorsque vous êtes prêt à clôturer une opportunité, vous pouvez changer son état dans la vue Détails des opportunités en Clôturée - Gagnée ou Clôturée - Perdue. Nous vous conseillons de clore une opportunité même si elle ne débouche pas sur une vente. Vous devrez fournir la raison de la fermeture de l'opportunité pour référence et rapport. Utilisez la zone Commentaires sur l'opportunité pour donner plus d'informations sur la raison pour laquelle la vente a été conclue ou perdue.

Pour clôturer une opportunité qui a abouti
  1. Ouvrez la vue Détails de l'opportunité à clôturer.
  2. Dans la zone État, cliquez sur le   , et sélectionnez Clôturé-Gagné dans la liste.
    La boîte de dialogue Clôturer l'opportunité apparaît.
  3. Le cas échéant, modifiez le montant réel de la vente.
  4. Le cas échéant, modifiez la Date de clôture réelle. Pour modifier la date, cliquez sur le Bouton  Agenda  , sélectionnez une nouvelle date, puis cliquez sur OK.
  5. Dans la zone Motif de la réussite, cliquez sur les points de suspensionpoints de suspension, sélectionnez un ou plusieurs éléments dans la liste, puis cliquez sur OK.
  6. Le cas échéant, cliquez sur le bouton Concur. remplacé Rechercher pour sélectionner le concurrent qui a perdu l'opportunité. Pour ce faire, sélectionnez un concurrent dans la boîte de dialogue Rechercher un concurrent puis cliquez sur OK. Répétez ce processus jusqu'à ce que vous ayez ajouté tous les concurrents remplacés.
  7. Dans la zone Commentaires, saisissez toute note supplémentaire.
  8. Cliquez sur OK.
Pour clôturer une opportunité qui n'a pas abouti
  1. Ouvrez la vue Détails de l'opportunité à clôturer.
  2. Dans la zone Etat, cliquez sur le   , et sélectionnez Clôturé-Perdu dans la liste.
  3. Le cas échéant, modifiez le montant réel de la vente.
  4. Le cas échéant, modifiez la Date de clôture réelle. Pour modifier la date, cliquez sur le Bouton  Agenda  , sélectionnez une nouvelle date, puis cliquez sur OK.
  5. Dans la zone Motif de la perte, cliquez sur les points de suspensionpoints de suspension, sélectionnez un ou plusieurs éléments dans la liste, puis cliquez sur OK.
  6. Si nécessaire, cliquez sur le bouton ConcurrentRechercher pour sélectionner le concurrent qui a perdu l'opportunité. Pour ce faire, sélectionnez un concurrent dans la boîte de dialogue Rechercher un concurrent puis cliquez sur OK.
  7. Dans la zone Commentaires, saisissez toute note supplémentaire.
  8. Cliquez sur Enregistrer.
