Suppression d'un produit d'une opportunité

Lorsque le client n'est plus intéressé par les produits ou si les produits ont été ajoutés par mégarde, vous pouvez supprimer ces produits de l'opportunité. Vous pouvez supprimer des produits à partir de la boîte de dialogue Insérer une opportunité ou de la vue Détails des opportunités. La suppression d'un produit supprime uniquement son association avec une opportunité donnée. Le produit n'est pas supprimé de la base de données.

Pour supprimer des produits à partir de la boîte de dialogue Insérer une opportunité

  1. Dans la grille Produits, sélectionnez le produit approprié, puis cliquez sur Supprimer. 
  2. Cliquez sur Supprimer.
  3. Cliquez sur OK. 

Pour supprimer des produits à partir de la vue Détails des opportunités

  1. Ouvrez la vue Détails Opportunité, puis cliquez sur l'onglet Produits. Si l'onglet n'apparaît pas, cliquez sur l'onglet Autres onglets.
  2. Cliquez sur le bouton Supprimer pour le produit à supprimer, puis sur OK pour confirmer la suppression.
