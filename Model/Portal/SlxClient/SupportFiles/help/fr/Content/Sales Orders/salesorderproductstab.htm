Onglet Produits du bon de commande

Utilisez cette vue pour ajouter, modifier ou supprimer des produits associés au bon de commande.

Procédure

Ouvrir cet onglet

  • Effectuez l'une des opérations suivantes :

  • Ouvrez la vue Détails du bon de commande et sélectionnez l'onglet Produits. Si l'onglet n'apparaît pas, cliquez sur Autres onglets.
  • Ajouter un bon de commande

Ajouter un produit

Ajouter ou modifier un produit personnalisé

Modifier les informations de produit

Modifier les informations de produit dans la grille
  • Sélectionnez l'enregistrement à modifier, faites vos modifications dans la grille, puis Cliquez sur Enregistrer. Pour annuler vos modifications, cliquez sur Réinitialiser avant de cliquer sur Enregistrer.
     

    Les colonnes ne peuvent pas toutes être modifiées dans la grille. Pour savoir si une colonne peut être modifiée, cliquez dans la grille. Si une zone blanche apparaît, la colonne est modifiable.

Supprimer un produit

Réordonner la liste par ordre numérique
  • Cliquez sur le bouton Renuméroter les éléments de ligne.

Cette vue contient les colonnes suivantes :

  • Modifier - Cliquez pour modifier les informations du produit.
    Cette option est uniquement disponible dans la vue Détails du bon de commande.
  • Utilisez la colonne Verrouillé ou cliquez sur le bouton Verrouiller tous les éléments ou Déverrouiller tous les éléments pour verrouiller ou déverrouiller le prix du produit. (Intégration de la comptabilité uniquement)
  • No. de ligne - Affiche le numéro de ligne attribué à chaque élément du bon de commande.
  • Produit - Affiche le nom du produit.
  • Famille - Affiche la famille du produit.
  • Niveau de prix - Affiche le niveau de prix utilisé pour ce produit.
  • Prix (devise de base) - Affiche le prix du produit. Cette colonne est basée sur le type de devise par défaut défini par votre administrateur.
  • Remise - Affiche le pourcentage de remise appliqué au produit.
  • Prix ajusté - Affiche le prix de base du produit moins la remise. Cette colonne est basée sur la devise par défaut définie par votre administrateur.
  • Quantité - Affiche la quantité numérique du produit ajoutée à l'opportunité. Vous permet de saisir une nouvelle quantité.
  • Unité : affiche l'échelle utilisée pour mesurer la quantité du produit. Par exemple, « à l'unité » ou « Pack de 10 ». (Intégration de la comptabilité uniquement)
  • Prix étendu (Devise de l'opportunité) - Affiche le total calculé du prix du produit multiplié par la quantité, moins toute remise éventuelle, et converti dans le type de devise de l'opportunité (Devises multiples uniquement)
  • Prix étendu (Devise de base) - affiche le total calculé du prix du produit multiplié par la quantité, moins toute remise éventuelle. Cette colonne est basée sur la devise par défaut définie par votre administrateur.
