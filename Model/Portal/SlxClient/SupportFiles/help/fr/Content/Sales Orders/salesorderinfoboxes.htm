Zones d'informations sur le bon de commande

Le tableau suivant décrit les zones d'informations par défaut.

Rectangle Description
Bon de commande ou devis État du bon de commande. Pour convertir un devis en bon de commande, copiez le devis et réglez le type sur bon de commande.
Compte Le compte associé au bon de commande. Pour trouver un compte, cliquez sur le bouton Rechercher. Ces informations sont obligatoires.
Responsable de comptes clients La personne qui, dans votre société, est responsable du compte. Cliquez sur le bouton Rechercher pour rechercher la personne ou l'équipe de votre entreprise qui gère le compte et ce bon de commande.
Opportunité L'opportunité associée au bon de commande. Pour trouver une opportunité, cliquez sur le bouton Rechercher.
Date de création

Date à laquelle le bon de commande a été créé.

Cette date apparaît uniquement sur la vue Détails du bon de commande et ne peut pas être modifiée.
Demandé par La personne ayant demandé le bon de commande. Cliquez sur le bouton Rechercher pour sélectionner un contact.
Date de la demande La date à laquelle le client a soumis le bon de commande. Pour effectuer une modification, cliquez sur le Bouton  Agenda  , sélectionnez une date.  
Date garantie La date à laquelle la commande doit être livrée au client. Pour effectuer une modification, cliquez sur le Bouton  Agenda  , sélectionnez une date.  
Date d'échéance La date à laquelle le bon de commande doit être livré au plus tard. Pour effectuer une modification, cliquez sur le Bouton  Agenda  , sélectionnez une date.  
BC client Le numéro du bon de commande client
Type État du bon de commande. Le type sélectionné ici et le type de bon de commande (bon de commande ou devis) peuvent être des valeurs différentes. Le type est une liste de sélection personnalisable qui peut contenir ou non des valeurs correspondant au type de bon de commande. Cliquez sur la flèche du menu déroulant et effectuez votre sélection dans la liste qui apparaît.
Status L'état du bon de commande. Cliquez sur    et sélectionnez un élément dans la liste. Si l'état devient Clôturé, tous les produits associés seront ajoutés en tant qu'actifs au compte.
Coût d'expédition Le coût d'expédition du bon de ommande. Entrez un nombre dans la zone.
Devise Le taux de change pour le bon de commande.Cliquez sur le bouton Rechercher pour ouvrir la recherche et sélectionner un taux de change
Taux de change Affiche le taux de change si vous avez sélectionné une devise.
Taux d'imposition Le taux d'imposition à appliquer au bon de commande. Saisissez le montant dans cette zone.
Commentaires Saisissez des remarques complémentaires concernant le bon de commande.
