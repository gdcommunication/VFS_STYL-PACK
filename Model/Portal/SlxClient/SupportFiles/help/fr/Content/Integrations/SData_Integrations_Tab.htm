Onglet Intégrations de la vue Configuration de la synchronisation SData

Utilisez l'onglet Intégrations pour voir la liste des intégrations qui synchronisent les données via SData. Vous pouvez aussi activer ou désactiver la synchronisation pour une intégration ou synchroniser manuellement d'un coup toutes les intégrations.

Procédure

Ouvrir l'onglet Intégrations
  • Dans la vue Configuration de la synchronisation SData, cliquez sur l'onglet Intégrations.
Activer ou désactiver une intégration pour la synchronisation
  1. Cochez la case Activé à côté d'une intégration pour activer ou désactiver la synchronisation.
    Cochez la case pour activer la synchronisation d'une intégration ou supprimez-la pour la désactiver.
  2. Cliquez sur Enregistrer.
Synchroniser d'un coup toutes les intégrations
  • Cliquez sur le bouton Synchroniser tous.
