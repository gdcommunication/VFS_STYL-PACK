Onglet Options d'intégration

Utilisez l'onglet Options pour afficher et modifier les options propres à une intégration. Ces options déterminent quelles informations sont synchronisées entre l'intégration et Saleslogix, et comment.

Les options d'intégration sont représentées par des paires nom-valeur. La plupart des intégrations incluent un ensemble par défaut de paires nom-valeur que vous pouvez modifier.

L'onglet Options est disponible uniquement pour les intégrations avec une méthode de synchronisation côté client ou côté serveur.

Procédure

Ouvrir l'onglet Options
  1. Ouvrez la vue Détails de l'intégration.
  2. Cliquez sur l'onglet Options.
Modifier des options
  1. Cliquez dans la colonne Valeur correspondant à l'option que vous voulez modifier.
    Utilisez la colonne Type de données pour identifier le type d'information à entrer comme valeur. Par exemple, pour un type de données Vrai/Faux, "Vrai" ou "Faux" est une valeur valide.
  2. Tapez la nouvelle valeur.
  3. Cliquez sur Enregistrer.
