Autorisation de services

Utilisez l'onglet Autoriser les services pour autoriser une connexion avec un service externe. Avant d'autoriser un service, votre administrateur doit configurer l'intégration et accorder l'accès aux utilisateurs. Ces options ne s'appliquent que lorsque l'utilisateur actuel est connecté à Client Web Saleslogix.
  • Il est possible que cette fonction ne soit pas disponible pour tous les utilisateurs. Les intégrations externes doivent être implémentées par votre administrateur.
  • Avant d'autoriser les services, vous devez autoriser les fenêtres pop-up. Consultez l'aide de votre navigateur pour obtenir des instructions spécifiques sur l'activation des pop-ups.

Pour autoriser un service

  1. Dans le menu Outils, cliquez sur Options.
  2. Cliquez sur l'onglet Autoriser les services.
  3. Cliquez sur Autoriser à côté du service que vous voulez autoriser.

  4. Pour procéder à l'autorisation, suivez les invites.
