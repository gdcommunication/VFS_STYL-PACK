Les champs clés sont sensibles à la casse. Par exemple, si vous avez un enregistrement pour Lee Hogan dans Saleslogix et un pour lee hogan dans Microsoft Outlook, les enregistrements ne sont pas égaux. Les deux sont ajoutés à chaque système lors de la synchronisation.

Correspondance de contacts

Les contacts sont appariés à l'aide d'un e-mail. Si une correspondance est introuvable, un nouvel enregistrement est créé. Pour éviter les enregistrements en double, Saleslogix recommande de s'assurer que les enregistrements de contact à synchroniser ont un e-mail unique dans les deux applications.

Si un e-mail correspondant exactement est introuvable, les scénarios suivants se produisent :

  • Dans Saleslogix, si l'e-mail est introuvable, un nouveau contact est créé.
  • Dans Saleslogix, si plusieurs enregistrements dans Saleslogix ont le même e-mail :
    • Saleslogix essaie de trouver une correspondance en utilisant le nom et le prénom du contact.
    • Si une correspondance est introuvable, un nouveau contact est créé dans Saleslogix.
  • Dans une application intégrée, si plusieurs enregistrements ont le même e-mail, le contact est associé au premier contact trouvé ayant un e-mail correspondant.
