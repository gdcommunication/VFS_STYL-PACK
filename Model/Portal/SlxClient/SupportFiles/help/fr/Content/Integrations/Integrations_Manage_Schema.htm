Gestion de schéma

Utilisez la boîte de dialogue Gérer schéma pour comparer le schéma de la base de données dans Saleslogix à votre schéma d'intégration pour garantir l'intégrité des données. Vous pouvez comparer les données de plusieurs ressources pour savoir si des différences de longueurs existent entre les données des systèmes.

En savoir plus sur la manière dont Saleslogix gère les différences de schéma...

Longueur Max : si la longueur des donnés synchronisées vers Saleslogix dépasse la LongueurMax d'un champ, la ligne entière est rejetée.

Décimales : si les donnés synchronisées vers Saleslogix contiennent un nombre de décimales supérieur à celui défini, la valeur est arrondie en fonction du nombre de décimales autorisé. Par exemple, si un champ de devise a la valeur 17,4, alors une valeur de 123,12348 est enregistrée sous la forme 123,1235 et une valeur de 123.12343 est enregistrée sous la forme 123,1234.

Procédure

Ouvrir la boîte de dialogue Gérer schéma
  • Dans la vue Détails de l'intégration, onglet Ressources, cliquez sur le bouton Gérer schéma.
Comparer le schéma pour les ressources
  1. Dans la boîte de dialogue Gérer schéma, zone Filtrer par ressource, sélectionnez le type de ressource.
  2. Cliquez sur Exécuter.
  3. Vérifiez les différences dans la grille.
    Notez toute information qui pourrait causer des problèmes d'intégrité des données. Par exemple, un champ du système de comptabilité peut n'accepter que trois caractères alors que le même champ dans Saleslogix accepte du texte libre.
  4. Répétez pour tous les types de ressources.
  5. Cliquez sur Terminer.
