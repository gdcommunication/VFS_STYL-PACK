Utilisation de la vue Configuration de synchronisation SData

Utilisez la vue Configuration de synchronisation SData pour configurer le flux Saleslogix utilisé pour synchroniser Saleslogix avec les systèmes intégrés.

Pour configurer SData

  1. Dans la barre de navigation, cliquez sur Configuration de synchronisation SData
  2. Configurez le flux Saleslogix. Utilisez les champs suivants pour configurer le flux Saleslogix. Le flux se met automatiquement à jour à mesure que vous entrez les données.
    1. Si votre URL SData utilise Secure Socket Layer, cochez la case Utiliser SSL (Secure Socket Layer).
    2. Dans la zone Serveur, tapez le nom du serveur pour votre URL SData.
    3. Dans la zone Port, tapez le numéro de port pour votre URL SData.
  3. Dans la zone Nom d'utilisateur, laissez "ADMIN" ou tapez le nom d'utilisateur correspondant à la base de données Saleslogix.

    L'authentification Windows doit être configurée pour l'utilisateur administratif. Pour les étapes détaillées, consultez la rubrique « Utilisation de l'authentification Windows » dans l'aide du module Administrateur.

  4. Dans le champ Mot de passe, saisissez le mot de passe correspondant à ce nom d'utilisateur.
  5. Dans la zone Nom d'affichage, laissez "Saleslogix" ou tapez le nom d'affichage que vous voulez voir représenter le flux SData.
  6. Cliquez sur le bouton Tester le lien et affichez les informations dans la zone État.
    Si le flux échoue au test, examinez les zones d'informations, apportez vos modifications et testez une nouvelle fois le lien.
  7. Cliquez sur Enregistrer.
