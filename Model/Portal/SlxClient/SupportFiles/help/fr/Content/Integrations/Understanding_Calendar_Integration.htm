Présentation de l’intégration des agendas

L'intégration de l'agenda vous permet de synchroniser votre agenda entre Saleslogix et une intégration prise en charge. Votre administrateur doit configurer et accorder une autorisation pour vous permettre de synchroniser des agendas avec une intégration.

During a synchronization cycle, Saleslogix compares fields in both systems, and makes changes to ensure both systems contain the same information. Only data in mapped fields between the applications is transferred. The data is compared and changes are made to ensure the information matches. For example, if a record has been added to one system and not to the other, that record is added to the incomplete system.

 La synchronisation est affectée par la sécurité intégrée des données associée à l'utilisateur connecté à Saleslogix. La quantité de données pouvant être transférée est limitée par les droits de sécurité de l'utilisateur connecté.

Comportement de synchronisation des agendas

  • Si le même enregistrement est mis à jour dans les deux systèmes durant le même cycle de synchronisation, la dernière modification a préséance.
  • La suppression d'une activité dans l'un ou l'autre système supprime l'activité des deux applications.
  • Responsables, membres et ressources d'activité :
    • Les informations concernant les membres et les ressources ne sont pas synchronisées.
    • Si l'utilisateur est membre ou responsable d'une activité, lorsque l'activité est synchronisée, l'utilisateur est listé comme responsable.
  • Les activités confirmées et non confirmées seront synchronisées.
  • Activités refusées :
    • Les activités refusées avant un cycle de synchronisation ne sont pas synchronisées.
    • Les activités qui sont refusées après un cycle de synchronisation sont supprimées de l'agenda de l'utilisateur dans les deux applications.
  • Les types d'activité dans Saleslogix peuvent être synchronisés avec différents types d'activité dans une application intégrée. Pour des détails, voir le tableau Présentation de l'échange de données d'agenda.

Pour des scénarios spécifiques d'intégration d'agendas, voir les scénarios de synchronisation d'agenda. 

Présentation de l'échange de données d'agenda

Le tableau répertorie comment l'agenda Saleslogix est synchronisé avec des intégrations externes :

Saleslogix Forme de synchronisation MS Outlook
Réunions Forme de synchronisation Rendez-vous
Appels téléphoniques Forme de synchronisation Rendez-vous
Activités personnelles Forme de synchronisation Rendez-vous, avec la mention « Privé »
Tâches à faire Forme de synchronisation Tâches
Événements Forme de synchronisation ne pas synchroniser
Appel téléphonique, réunion ou activité personnelle sans horaire défini Forme de synchronisation Evénement d'une journée entière

Procédure

  • Définition des options de l'agenda
  • Synchronisation manuelle de l'agenda et des contacts

Il est déconseillé de synchroniser simultanément le client distant avec la base de données du bureau principal et Microsoft Outlook. Cela réduit considérablement la vitesse de la synchronisation et provoque des problèmes de mémoire. Saleslogix recommande vous recommande de vous synchroniser avec MS Outlook avant ou après avoir synchronisé votre base de données distante.
