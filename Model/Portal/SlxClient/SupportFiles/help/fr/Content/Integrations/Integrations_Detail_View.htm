Utilisation de la vue Détails de l'intégration

L'administrateur système et les utilisateurs dotés d'un rôle approprié peuvent accéder à ces fonctionnalités.

Utilisez la vue Détails pour afficher, ajouter ou modifier des informations sur une intégration sélectionnée. La vue Détails est composée de zones d'informations, d'onglets et d'une section centrale définie par l'utilisateur.

Les informations d'intégration visibles et modifiables peuvent être différentes, selon le type d'authentification et l'intégration en question.

Procédure

Ouvrir la vue Détails de l'intégration
  • Sur la barre de navigation, cliquez sur Intégrations pour ouvrir la vue Liste Intégrations. Si nécessaire, utilisez la Recherche pour localiser une intégration, puis cliquez sur le nom de l'intégration.

Utiliser la fonction Rechercher pour rechercher des enregistrements 

Passer en vue Liste
  • Dans la barre d'outils, cliquez sur le bouton Vue Liste, ou cliquez sur l'onglet du groupe actif.

Afficher un groupe

Zones d'informations d'intégration

Les zones d'informations d'intégration contient les informations principales sur l'intégration. Selon l'intégration, vous pouvez afficher ou modifier les informations si nécessaire. Certaines informations sont non modifiables.

onglets des vues Détails

Les onglets qui s'affichent dépendent du type d'authentification d'intégration et des paramètres de synchronisation. Cliquez sur un lien pour voir davantage d'informations sur chaque onglet.
