Utilisation de la vue Liste des intégrations

L'administrateur système et les utilisateurs dotés d'un rôle approprié peuvent accéder à ces fonctionnalités.

La vue Liste des intégrations affiche toutes les intégrations auxquelles vous avez accès. Cette liste inclut toutes les intégrations émises par Saleslogix, de même que toutes les intégrations personnalisées. Utilisez les options de groupes et de recherche pour restreindre la liste des enregistrements qui s'affiche.

Procédure

Ouvrir la vue Liste des intégrations
  • Sur la barre de navigation, cliquez sur Intégrations.

Travailler avec des enregistrements dans une grille

Actualiser les informations de la liste
  • Cliquez sur le bouton Actualiser.

Filtrer ma liste

Utiliser la fonction Rechercher pour rechercher des enregistrements 

Ouvrir la vue Détails de l'intégration
  • Cliquez sur le nom de l'intégration pour ouvrir la vue Détails de l'intégration.

Afficher un groupe

Utilisation du menu contextuel des groupes

Utiliser les tâches courantes

Dans le Volet des tâches, sous Tâches courantes, vous pouvez :

  • Ajouter au groupe.
  • Supprimer du groupe.
  • Enregistrer les enregistrements en tant que groupe
  • Exporter vers un fichier
