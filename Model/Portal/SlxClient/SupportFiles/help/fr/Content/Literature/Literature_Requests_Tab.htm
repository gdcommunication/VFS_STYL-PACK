Onglet Demandes de documentation

L'onglet Demandes de documentation figure dans les vues Détails du compte et Détails du contact et contient toutes les demandes de documentation concernant le compte ou le contact. L'onglet Demandes de documentation présente les informations suivantes : date de la demande, date de l'envoi et objet de la demande.

Procédure

Afficher une demande de documentation
  1. Dans les vues Détails du contact ou Détails du compte, cliquez sur l'onglet Demandes de documentation. Si l'onglet n'apparaît pas, cliquez sur l'onglet Autres onglets.
  2. Cliquez sur Description pour en savoir plus sur la demande de documentation sélectionnée.
    La vue Détails de la demande de documentation s'ouvre.
Planifier une demande de documentation
  1.  Cliquez sur le bouton Planifier une demande de documentation..
  2. Suivez les étapes permettant de planifier une demande de documentation.
