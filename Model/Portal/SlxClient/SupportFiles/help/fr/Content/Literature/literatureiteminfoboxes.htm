Zone d'informations des éléments de documentation

Le tableau suivant décrit les zones d'informations par défaut.

 

Rectangle Description/Action
N° d'article

L'ID du document attribué par votre responsable de documentation.

Vous n'êtes pas obligé de renseigner la zone Numéro d’élément. Faites le uniquement si votre société a attribué un numéro à chaque type de documentation.
Nom Le nom du document.
Auteur L'auteur du document.
Famille Cliquez sur    et sélectionnez un élément dans la liste.
Coût Le coût par unité de cette fonctionnalité.
En stock

La quantité actuelle en stock.

Ce nombre diminue à chaque nouvelle demande de documentation traitée. Si vous ne renseignez pas la zone Disponible, la quantité en stock ne change pas. Si vous tapez 0 dans la zone disponible, la quantité disponible diminue et devient négative.
Qté de réapprovisionnement

La quantité minimale pour vous avertir que vous devez réapprovisionner votre stock.

Si la quantité disponible est inférieure ou égale à la quantité de réapprovisionnement, un message s'affiche pour vous avertir que vous devez réapprovisionner votre stock.
File Si le document se présente sous la forme d'un fichier, tapez son chemin d'accès dans la zone Fichier.
Expire le

Cliquez sur le bouton Agenda pour sélectionner une date puis cliquez sur OK.
