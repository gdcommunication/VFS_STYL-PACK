Liaison d'adresses de contacts

L'étape de liaison d'adresses de contacts vous permet d'associer les contacts que vous avez liés dans l'écran précédent aux informations d'adresse lors de la fusion d'enregistrement.

Pour lier une adresse pour des enregistrements fusionnés

  1. Ouvrez l'assistant de liaison ou l'assistant de fusion d'enregistrements et passez aux écrans précédents.
  2. Remplissez l'écran Adresses de contacts liées.
    ZoneDescription/Action
    Contacts liés

    Affiche les contacts liés de l'écran précédent.

    Cliquez sur Sélectionner pour sélectionner les détails d'un contact.

    Adresses sources non liées issues de

    Affiche les adresses non liées pour le contact source sélectionné.

    Cliquez sur Sélectionner pour sélectionner les détails de l'adresse.

    Adresses cibles non liées issues de

    Affiche les adresses non liées pour le contact cible sélectionné.

    Cliquez sur Sélectionner pour sélectionner les détails de l'adresse.

    Bouton Lier

    Sélectionnez une adresse non liée de chaque liste, puis cliquez sur Lier pour créer une connexion.

    Adresses de contacts liées

    Affiche toutes les adresses de contacts liées.

    • Cliquez sur Dissocier pour supprimer le lien proposé et en créer un nouveau.
  3. Cliquez sur l'option Fusionner.
