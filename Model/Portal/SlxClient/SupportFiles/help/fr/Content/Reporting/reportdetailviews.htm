Affichage et impression de rapports de vue Détails

Certaines vues Détails contiennent un rapport détaillé que vous pouvez afficher ou imprimer. Vous pouvez afficher et imprimer des rapports détaillés depuis les vues Détails du contact, Détails du compte, Détails de l'opportunité, Détails des bons de commande, Détails du ticket et Détails du défaut.

Les rapports détaillés ne sont pas disponibles pour toutes les vues Détails.

Pour visualiser un rapport

  1. Ouvrez la vue Détails de l'enregistrement que vous souhaitez afficher ou imprimer.
  2. Dans le Volet des tâches, sous Tâches courantes, cliquez sur Rapport détaillé.
  3. Sélectionnez les options Valeurs à inclure dans le rapport.
  4. Cliquez sur Terminer.
  5. Ouvrez le rapport à partir d'un des endroits suivants :
    • champ de message Tâche terminée
    • page Notifications de tâches
    • onglet Historique des rapports - vue Liste
  6. Acrobat Reader s'ouvre et affiche le rapport que vous avez sélectionné.

Pour imprimer un rapport détaillé

  1. Ouvrez le rapport à partir d'un des endroits suivants :
    • champ de message Tâche terminée
    • page Notifications de tâches
    • onglet Historique des rapports - vue Liste
  2. Acrobat Reader s'ouvre et affiche le rapport que vous avez sélectionné.
  3. Cliquez sur Imprimer.
    Si le rapport ne s'imprime pas correctement, vous devrez peut-être le reformater pour votre imprimante. L'orientation du rapport aura la priorité sur l'orientation définie dans les paramètres de l'imprimante.
