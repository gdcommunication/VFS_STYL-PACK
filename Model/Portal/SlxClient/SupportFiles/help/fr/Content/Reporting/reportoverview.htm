Présentation des rapports

La vue Liste des rapports offre une liste de rapports types prédéfinis pour vous permettre d'afficher et d'analyser les informations dans la base de données Saleslogix. Vous pouvez exécuter des rapports liés aux vues principales telles que Contact, Compte, Opportunité, etc. Si vous le préférez, vous pouvez modifier le rapport en fonction de vos besoins. Les rapports spécifiques à votre disposition sont déterminés par votre administrateur système.

Saleslogix Web Les rapports vous permettent de :

  • Faire votre sélection sur une liste de rapports types prédéfinis et d'appliquer des filtres pour réduire le champ de recherche et les résultats
  • Programmer un rapport à exécuter à une date ultérieure une fois ou de manière récurrente
  • Voir la liste des rapports et les heures auxquelles ils doivent s'excuter.
  • Afficher, imprimer et stocker des rapports.

Pour ouvrir la vue Rapports

  • Dans la barre de navigation, cliquez sur Rapports.

Pour exécuter des rapports dans un état déconnecté, le Client Web hors ligne doit être exécuté sur le serveur Windows 2003 avec IIS. Le Saleslogix Webserveur Reporting doivent également être installés et configurés. Si ces conditions ne sont pas remplies, les utilisateurs du client Web déconnectés ne peuvent exécuter Web Reporting que s'ils sont connectés au serveur Web Reporting hôte. Ces utilisateurs verront des rapports qui représentent les données de la base de données hôte et non de leur base de données distante.
