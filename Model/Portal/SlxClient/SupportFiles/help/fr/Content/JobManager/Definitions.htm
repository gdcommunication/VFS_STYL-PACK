Définitions

L'onglet Définitions contient la liste de types de tâche prédéfinis. Sélectionnez un type de tâche parmi ceux listés et utilisez l'assistant de planification pour définir une heure et une récurrence d'exécution.

Remarque : De nouveaux types de tâche sont créés dans le Saleslogix Application Architect.

Pour créer un planning pour une tâche

1. Sélectionnez la tâche, cliquez le bouton droit de la souris, puis cliquez sur Créer un planning.

2. Dans la boîte de dialogue Exécuter/planifier une tâche, apportez vos modifications.

3. Cliquez sur OK.
