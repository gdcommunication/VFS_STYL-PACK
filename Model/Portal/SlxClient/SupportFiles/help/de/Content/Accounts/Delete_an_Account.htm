Löschen einer Firma

Wenn Sie eine Firma löschen, löschen Sie gleichzeitig die Historie von Kontakten, Firmen und Verkaufschancen sowie Notizen und offene Aktivitäten.

So löschen Sie eine Firma

  1. Klicken Sie auf der Navigationsleiste auf Firmen.
  2. Suchen Sie die zu löschende Firma. 
  3. Klicken Sie in der Ansicht Firmendetail auf die Schaltfläche Löschen. 
  4.  Klicken Sie auf "OK".
