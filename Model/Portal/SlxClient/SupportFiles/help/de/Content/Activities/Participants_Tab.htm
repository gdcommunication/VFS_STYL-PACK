Teilnehmer Registerkarte

Verwenden Sie die Registerkarte "Teilnehmer", um weitere Kontakte und Interessenten hinzuzufügen, die einer Aktivität zugeordnet sind.

Wenn ein Kontakt oder Interessent auf der Registerkarte "Allgemein" hinzugefügt wird, wird der Kontakt oder Interessent standardmäßig und automatisch als Hauptteilnehmer zur Registerkarte "Teilnehmer" hinzugefügt. Wenn Sie auf der Registerkarte "Teilnehmer" einen anderen Hauptteilnehmer auswählen, wird die Registerkarte "Allgemein" mit dem neuen Hauptkontakt oder -interessenten aktualisiert. Sowohl die Kontakte als auch die Interessenten verbleiben auf der Registerkarte "Teilnehmer", sofern sie nicht mit der Schaltfläche "Löschen" entfernt werden.

Wenn Sie einen Kontakt, der einem Saleslogix-Benutzer zugeordnet ist, zur Registerkarte Teilnehmer hinzufügen, wird der zugeordnete Benutzer automatisch zur Registerkarte Verfügbarkeit hinzugefügt, damit Sie sehen können, ob der Benutzer frei oder gebucht ist. Wenn ein Saleslogix einem Kontakt zugeordneter Benutzer zur Registerkarte Verfügbarkeit hinzugefügt wird, dann wird der Kontakt zur Registerkarte "Teilnehmer" hinzugefügt.

Wenn ein Saleslogix einem Kontakt zugeordneter Benutzer zur Registerkarte Verfügbarkeit hinzugefügt wird, dann wird der Kontakt zur Registerkarte Teilnehmer hinzugefügt. Wenn Sie einen Kontakt, der einem Saleslogix-Benutzer zugeordnet ist, zur Registerkarte Teilnehmer hinzufügen, wird der zugeordnete Benutzer automatisch zur Registerkarte "Verfügbarkeit" hinzugefügt, damit Sie sehen können, ob der Benutzer frei oder gebucht ist.

Wenn Sie einen Kontakt oder Interessenten zur Registerkarte "Teilnehmer" hinzufügen, erscheint die Aktivität in den zugeordneten Kontakt- und Firmendetailansichten oder in der Interessenten-Detailansicht auf der Registerkarte "Aktivitäten".

Vorgehensweise

Öffnen der Registerkarte Teilnehmer
  • Planen oder bearbeiten Sie ein Meeting oder einen Anruf, und klicken Sie auf die Registerkarte "Teilnehmer".
Hinzufügen von einem oder mehreren Kontakten
  1. Klicken Sie auf der Registerkarte "Teilnehmer" auf die Schaltfläche Kontakt hinzufügen.
  2. Verwenden Sie die Suchfunktion, um Kontakte zu finden, die Sie zur Aktivität hinzufügen möchten.
    Mit den Schaltflächen "Suchen nach" und "Suchen" können Sie die Liste eingrenzen.
  3. Wählen Sie einen oder mehrere Kontakte aus, und klicken Sie auf Ausgewählte hinzufügen.
  4. Klicken Sie auf Schließen, wenn Sie fertig sind.
Hinzufügen von einem oder mehreren Interessenten
  1. Klicken Sie auf der Registerkarte "Teilnehmer" auf die Schaltfläche Interessenten hinzufügen.
  2. Verwenden Sie die Suchfunktion, um Interessenten zu finden, die Sie zur Aktivität hinzufügen möchten.
    Mit den Schaltflächen "Suchen nach" und "Suchen" können Sie die Liste eingrenzen.
  3. Wählen Sie einen oder mehrere Interessenten aus, und klicken Sie auf Ausgewählte hinzufügen.
  4. Klicken Sie auf Schließen, wenn Sie fertig sind.
Hinzufügen von Kontakten und Interessenten
  1. Klicken Sie auf der Registerkarte "Teilnehmer" auf die Schaltfläche Teilnehmer hinzufügen.
  2. Verwenden Sie das Dialogfeld Teilnehmer suchen, um Kontakte oder Interessenten zu suchen, die Sie zur Aktivität hinzufügen möchten.
  3. Geben Sie die ersten Buchstaben des Vor- oder Nachnamens der Person ein, und klicken Sie auf Suchen.
  4. Wählen Sie einen oder mehrere Kontakte oder Interessenten aus, und klicken Sie auf Ausgewählte hinzufügen.
  5. Wiederholen Sie die Schritte 4 und 5, bis Sie alle Kontakte oder Interessenten ausgewählt haben, die Sie zur Aktivität hinzufügen möchten.
  6. Klicken Sie auf Schließen, wenn Sie fertig sind.
Ändern des Hauptkontaktes oder -interessenten
  • Markieren Sie im Raster das Kontrollkästchen Haupt neben dem Kontakt oder Interessenten, den Sie zum Hauptkontakt oder -interessenten machen möchten.
    Die Registerkarte "Allgemein" wird aktualisiert und enthält den neuen Hauptkontakt oder -interessenten.
Zuweisen einer Rolle zu einem Teilnehmer
  1. Klicken Sie im Raster in die Spalte Rolle für den Teilnehmer, den Sie bearbeiten möchten.
  2. Geben Sie eine Rolle ein, oder klicken Sie auf den Dropdown-Pfeil, und wählen einen Eintrag in der Liste aus.
Kennzeichnen als Teilnehmer

Von der Registerkarte "Allgemein" hinzugefügte Kontakte und Interessenten werden mit nicht markiertem Kontrollkästchen "Ist Teilnehmer" zur Registerkarte "Teilnehmer" hinzugefügt, um darauf hinzuweisen, dass sie nur der Aktivität zugeordnet sind. Von der Registerkarte "Teilnehmer" hinzugefügte Kontakte und Interessenten werden mit markiertem Kontrollkästchen "Ist Teilnehmer" hinzugefügt, um darauf hinzuweisen, dass sie Teilnehmer des Meetings sind. Sie können die Spalte "Ist Teilnehmer" für jeden Kontakt und Interessenten bearbeiten.

  • Markieren Sie das Kontrollkästchen Ist Teilnehmer, um darauf hinzuweisen, dass der Kontakt oder Interessent nicht nur der Aktivität zugeordnet ist, sondern auch ein Teilnehmer ist.
  • Wenn der Kontakt oder Interessent der Aktivität zugeordnet ist, jedoch nicht an der Aktivität teilnehmen wird, entfernen Sie die Markierung im Kontrollkästchen Ist Teilnehmer.
Entfernen eines Kontakts oder Interessenten
  1. Wählen Sie einen oder mehrere Datensätze in der Liste aus, und klicken Sie auf die Schaltfläche Löschen .
  2. Klicken Sie auf "OK".
