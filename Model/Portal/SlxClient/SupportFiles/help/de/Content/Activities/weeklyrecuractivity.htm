Sich wiederholende Aktivitäten - Wöchentlich

Sie können festlegen, ob eine Aktivität Einmal, Täglich, "Wöchentlich", Monatlich oder Jährlich stattfinden soll.

So planen Sie eine wöchentliche periodische Aktivität

  1. Planen oder bearbeiten Sie eine Aktivität.
  2. Klicken Sie auf die Registerkarte Sich wiederholend.
  3. Wählen Sie unter Diese Aktivität findet statt... die Option Wöchentlich aus.
  4. Legen Sie fest, wie oft die Aktivität wiederholt werden soll.
    • Wenn die Aktivität an einem bestimmten Tag in einer bestimmten Woche wiederholt werden soll, wählen Sie die Option Alle X Wochen am aus und geben Sie die Anzahl der Wochen zwischen den Wiederholungen ein.Wählen Sie anschließend den Tag der Woche aus, an dem die Aktivität wiederholt werden soll. 
    • Wenn die Aktivität nur nach Abschluss der vorherigen Aktivitäten wiederholt werden soll, wählen Sie die Option Alle X Wochen nach Abschluss aus, und geben Sie die Anzahl der Wochen zwischen den Wiederholungen ein.
  5. Klicken Sie im Feld Sich wiederholende Aktivität starten auf die  Kalender  , wählen Sie ein Datum aus, und klicken Sie dann auf OK.

  6. Führen Sie einen der folgenden Schritte aus:

    • Geben Sie in das Feld Ende nach ein, wie oft die Aktivität sich wiederholen soll.
    • Klicken Sie unter Wiederholung beenden auf die  Kalender  , wählen Sie ein anderes Datum aus, und klicken Sie dann auf OK. 
  7. Wenn Sie alle Aktivitätsänderungen vorgenommen haben, klicken Sie auf OK.
