Planen oder Bearbeiten einer Aktivität

Verwenden Sie die Aktivitätsdetailansicht, um eine Aktivität hinzuzufügen oder zu bearbeiten oder um ausführliche Informationen zu einer Aktivität anzuzeigen, wie Datum, Zeit, Dauer und Notizen. Sie können eine Aktivität nach dem Planen auch abschließen oder löschen.

Die Kalendersicherheitseinstellungen bestimmen, wessen Aktivitäten Sie planen, bearbeiten, akzeptieren/ablehnen oder löschen können. Sie können auch Aktivitäten anzeigen, die mit Kontakten, Interessenten, Firmen, Verkaufschancen oder Tickets verknüpft sind, auf die Sie Zugriff haben.

Vorgehensweise

Planen einer Aktivität

  1. Führen Sie einen der folgenden Schritte aus:

    • Klicken Sie im Menü Planen auf den zu planenden Aktivitätstyp ("Anruf", "Meeting", "Aufgabe" oder "Persönliche Aktivität").
    • Klicken Sie in der Detailansicht Kontakt, Firma, Verkaufschance oder Ticket auf die Registerkarte Aktivitäten und anschließend auf die Schaltfläche Anruf planen, Meeting planen oder Aufgabe planen. Falls die Registerkarte nicht angezeigt wird, klicken Sie auf die Registerkarte Weitere Registerkarten.
  2. Klicken Sie im Feld Betreff auf  Dropdown-Pfeil , und wählen Sie einen Eintrag in der Liste aus.
  3. Geben Sie in das Feld Standort einen Standort ein, oder verwenden Sie die Registerkarte Verfügbarkeit, um eine Standortressource auszuwählen.
  4. Klicken Sie im Feld Startzeit auf die  Kalender  , um das Datum und die Uhrzeit der Aktivität auszuwählen, und klicken Sie dann auf OK.

  5. Führen Sie einen der folgenden Schritte aus:

    • Aktivieren Sie das Kontrollkästchen Ohne Zeitangabe, um eine Aktivität zu erstellen, die an einem bestimmten Termin stattfindet, aber keine spezifische Uhrzeit und Dauer aufweist.
      Aktivitäten ohne Zeitangabe werden in der Tagesansicht des Kalenders angezeigt.
    • Geben Sie in das Feld Dauer einen Wert ein, oder klicken Sie auf den den Dropdown-Pfeil, und wählen einen Eintrag in der Liste aus.
  6. Aktivieren Sie das Kontrollkästchen Alarm, um einen Benachrichtigungsalarm vor Beginn der Aktivität einzurichten. Geben Sie eine Zeitdauer ein bzw. wählen Sie sie aus.
  7. Aktivieren Sie das Kontrollkästchen Automatisch auf nächsten Tag ausdehnen, um die Aktivität automatisch auf den nächsten Tag auszudehnen, wenn sie nicht am geplanten Termin abgeschlossen wird. Diese Option ist nur verfügbar, wenn "Ohne Zeitangabe" ausgewählt ist.

    Sich wiederholende Aktivitäten können nicht ausgedehnt werden. Wenn Sie diese Option auswählen und dann Wiederholungsoptionen festlegen, wird die automatische Ausdehnung auf den nächsten Tag aufgehoben.

  8. Wählen Sie ggf. Kontakt oder Interessent aus.
  9. Klicken Sie auf die Schaltfläche Suchen neben dem entsprechenden Feld (Name, Firma, Verkaufschance, Ticket oder Unternehmen), um die Suchfunktion aufzurufen. Suchen Sie den Datensatz, den Sie der Aktivität zuordnen möchten, und klicken Sie auf OK. (Wenn Sie eine persönliche Aktivität planen, fahren Sie mit dem nächsten Schritt fort.)
    Der Kontakt oder Interessent wird als Hauptkontakt oder -interessent zur Registerkarte "Teilnehmer" hinzugefügt. Um die Zuordnung des Kontakts oder Interessenten mit der Aktivität beizubehalten, aber einen anderen Hauptkontakt oder -interessenten auszuwählen, klicken Sie auf die Schaltfläche Suchen, und wählen Sie einen anderen Kontakt oder Interessenten aus.

    Klicken Sie auf die Schaltfläche Entfernen neben dem entsprechenden Feld, um eine Firma, eine Verkaufschance oder ein Ticket aus der Aktivität zu entfernen. Kontakte oder Interessenten werden nicht aus der Aktivität entfernt, wenn Sie auf die Schaltfläche Entfernen klicken. Kontakte und Interessenten müssen aus der Registerkarte "Teilnehmer" entfernt werden.

  10. Nach Auswählen eines Kontaktes oder Interessenten können Sie den Mauszeiger über den Datensatznamen bewegen, um eingeschränkte Detailinformationen anzuzeigen.
  11. Wählen Sie im Feld Priorität die Wichtigkeit der Aktivität aus.
  12. Wählen Sie im Feld Kategorie die Kategorie der Aktivität aus.
  13. Wählen Sie im Feld Organisator einen Organisator aus, wenn Sie nicht selbst der Organisator der Aktivität sind.Klicken Sie dazu auf die Schaltfläche  Suchen, und verwenden Sie die Suchfunktion. Nicht alle Benutzer stehen zur Verfügung. Sie können einen Benutzer nur als Organisator der Aktivität auswählen, wenn Sie Zugriff auf seinen Kalender haben.
  14. Geben Sie im Bereich Notizen ggf. Notizen zur Aktivität ein.
  15. Click the Availability tab to select and check the availability of the members and resources participating in the activity.

    Sie können nur Mitglieder oder Ressourcen zu Anrufen und Meetings hinzufügen.
  16. Klicken Sie auf die Registerkarte "Teilnehmer", um weitere Kontakte und Interessenten einzubeziehen.
  17. Klicken Sie auf die Registerkarte Wiederkehrend, um auszuwählen, wie oft die Aktivität stattfinden soll (Einmal, Täglich, Wöchentlich, Monatlich oder Jährlich).
  18. Klicken Sie auf die Registerkarte Anlagen, um ein Dokument oder einen URL hinzuzufügen oder ein Dokument oder einen URL anzuzeigen.

    Wenn ein Aktivitätsmitglied Zugriffsrechte für den Kalender des Organisators der Aktivität hat, kann das Mitglied Anlagen hinzufügen oder bearbeiten. Ist dies nicht der Fall, kann das Mitglied Anlagen nur anzeigen, kopieren und speichern.

  19. Klicken Sie auf "OK".
Bearbeiten einer Aktivität
  1. Führen Sie einen der folgenden Schritte aus:
    • Öffnen Sie die Listenansicht Aktivitäten, suchen Sie die zu bearbeitende Aktivität, und klicken Sie dann auf den Hyperlink Typ der Aktivität.
    • Klicken Sie in der Ansicht Neuigkeiten auf die Registerkarte Aktivitäten und anschließend auf das Symbol des Typs der zu bearbeitenden Aktivität.
  2. Wenn Sie eine periodische Aktivität aktualisieren, wählen Sie bei entsprechender Aufforderung Bearbeiten Sie dieses Vorkommen der Aktivität aus, um nur die ausgewählte Wiederholung zu ändern, oder Bearbeiten Sie alle Vorkommen dieser Aktivität, um die Änderung auf alle Wiederholungen der Aktivität anzuwenden, und klicken Sie anschließend auf Fortfahren.

    		3.

    Die Registerkarte "Sich wiederholend" steht nicht zur Verfügung, wenn Sie die Option "Bearbeiten Sie dieses Vorkommen der Aktivität" auswählen.
  3. Bearbeiten Sie die Informationen zur Aktivität.
    Sie können den Mauszeiger über einen Kontakt, eine Firma oder einen Interessenten bewegen, um eingeschränkte Detailinformationen anzuzeigen.
  4. Klicken Sie auf "OK".

    		5.

    Sie können den Benutzer einer persönlichen Aktivität nicht ändern.
