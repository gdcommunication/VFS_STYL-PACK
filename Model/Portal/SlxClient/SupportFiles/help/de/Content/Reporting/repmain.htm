Berichtsvorschau

Ihre fertiggestellten Berichte werden auf der Registerkarte "Berichtshistorie" gespeichert.

Sie können nur Berichte anzeigen, die für Sie oder ein Team freigegeben sind, dem Sie angehören. Sie können keine Berichte in Saleslogix Web Client speichern.

Damit Sie Berichte exportieren oder drucken können, müssen Sie die Anzeige von Popup-Fenstern erlauben.

Vorgehensweise

Anzeigen oder Drucken eines Detailberichts aus einer Detailansicht

Anzeigen oder Drucken eines Berichts über die Registerkarte "Berichtshistorie"
  1. Klicken Sie auf der Navigationsleiste auf Berichte, und klicken Sie auf die Registerkarte Historie.
  2. Suchen Sie den gewünschten Bericht. Sie können nach Rasterspalten-Überschriften sortieren.
  3. Klicken Sie auf den Bericht.
  4. Der Bericht wird in der nativen Ansicht des Dateityps angezeigt. Beispiel: Acrobat Reader für PDF-Formate.
Vorschau auf einen Bericht im Viewer
  1. Wählen Sie den Bericht aus, zu dem Sie eine Vorschau anzeigen möchten, und klicken Sie dann auf Berichte anzeigen. Der Bericht wird im Report Viewer angezeigt. Je nach ausgewähltem Bericht und der Anzahl der erstellten Ergebnisse kann eine Strukturansicht auf der linken Seite des Fensters angezeigt werden. Diese Strukturansicht führt die Seiten im Bericht in alphanumerischer oder chronologischer Reihenfolge auf.
  2. Um eine bestimmte Seite des Berichts anzuzeigen, klicken Sie auf das Plus-Zeichen, um die Liste zu erweitern, und anschließend in der Verzeichnisstruktur auf einen Namen.
