Tipps für neue Benutzer

Falls Sie den Saleslogix Web Client oder das SalesLogix-Kundenportal-Kundenportal zum ersten Mal verwenden, beachten Sie bitte folgende Tipps:

  • Nachdem Sie eine Änderung vorgenommen haben, klicken Sie auf (Speichern).
  • Jedes Mal, wenn Sie das Symbol (Suchen) sehen, verwenden Sie die Suchfunktion, um einen gesuchten Datensatz zu finden.
  • Saleslogix kann exakt an Ihre Anforderungen angepasst werden. Ihr Systemadministrator kann viele Ansichten und Funktionen von Saleslogix anpassen und Ihre persönlichen Zugriffsrechte festlegen. Aus diesem Grund können Sie möglicherweise nicht auf alle Funktionen zugreifen, die im Online-Hilfesystem erklärt werden.Es ist auch möglich, dass einige Ansichten in Ihrer Anwendung anders aussehen.
  • Bei Fragen können Sie im Hilfesystem schnell Antworten finden.Das Hilfesystem enthält Erklärungen zu Funktionen sowie Links zu verwandten Themen. Das Hilfesystem kann durch Klicken auf das Symbol (Hilfe) in allen Bildschirmen geöffnet werden. Die für den jeweiligen Bildschirm geeignete Hilfe wird angezeigt. Klicken Sie innerhalb der Hilfe auf "Inhaltsverzeichnis", "Index" oder "Suchen", um nach einem bestimmten Hilfethema zu suchen.
