Einführung: Saleslogix for Web

Mit Saleslogix können Sie alle Aspekte des Vertriebszyklus über das Web verwalten. Sie können nicht nur Kontakte, Firmen, Interessenten, Verkaufschancen, Tickets und Aktivitäten verfolgen, sondern haben auch Zugriff auf Tools für Berichte, die Prozessautomatisierung und die Erstellung von Serien-E-Mails, -Faxen, und -Briefen. Mithilfe zusätzlicher Sicherheitsberechtigungen steht Ihnen ggf. ein Zugriff auf Tools für die Verwaltung von Benutzern, Teams, Abteilungen und Auswahllisten zur Verfügung.

Abhängig von den durch Ihren Systemadministrator installierten Komponenten können Sie auf den Saleslogix Web Client über das Netzwerk oder über ein Web-Phone zugreifen.

In diesem Hilfesystem wird erklärt, wie Sie das Saleslogix Web Client verwenden. Je nach Ihren Zugriffsrechten und abhängig davon, ob Ihre Installation einen speziell angepassten Saleslogix Web Client verwendet, ist Ihr Arbeitsbereich möglicherweise anders beschaffen.

Weitere Informationen erhalten Sie von Ihrem Systemadministrator.
