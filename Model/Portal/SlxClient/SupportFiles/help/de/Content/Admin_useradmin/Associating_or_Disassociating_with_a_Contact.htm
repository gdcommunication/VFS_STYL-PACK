Zuordnen zu einem Kontakt oder Aufheben der Zuordnung

Wenn Ihre Saleslogix-Implementierung das Erstellen eines Kontaktes für jeden Saleslogix-Benutzer beinhaltet, können Sie den Benutzer und Kontakt problemlos aus der Listenansicht "Benutzer" zuordnen.

Der Systemadministrator und Benutzer mit der entsprechenden Rolle können auf diese Funktionen zugreifen.

Vorgehensweise

Ordnen Sie einen Benutzer einem Kontakt zu,
  1. Klicken Sie auf der Navigationsleiste auf Administration., und klicken Sie dann auf Benutzer.
  2. Wählen Sie den Benutzer aus, den Sie einem Kontakt zuordnen möchten.
  3. Klicken Sie im Aufgabenbereich unter Benutzeraufgaben auf Zu Kontakt zuordnen.
  4. Klicken Sie im Dialogfeld Benutzer zu Kontakt zuordnen auf Kontakt Schaltfläche  Suchen.
  5. Verwenden Sie die Suchfunktion, um den Kontakt zu suchen, den Sie dem Benutzer zuordnen möchten, und klicken Sie dann auf OK.
  6. Klicken Sie auf "OK".
Heben Sie die Zuordnung eines Benutzers mit einem Kontakt auf,
  1. Klicken Sie auf der Navigationsleiste auf Administration., und klicken Sie dann auf Benutzer.
  2. Wählen Sie einen oder mehrere Benutzer aus, deren Zuordnung zu Kontakten Sie aufheben möchten.
  3. Klicken Sie im Aufgabenbereich unter Benutzeraufgaben auf Zuordnung zu Kontakt aufheben.
  4. Klicken Sie auf "OK".
