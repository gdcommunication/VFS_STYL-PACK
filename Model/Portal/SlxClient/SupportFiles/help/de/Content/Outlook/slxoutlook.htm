Verwenden der Outlook-Integration

   Mithilfe der Outlook-Integration können Sie   Outlook-E-Mail-Nachrichten als Saleslogix Historieneinträge speichern. Beispielsweise sind folgende Aktionen möglich:

  • Verfassen Sie eine E-Mail-Nachricht in Outlook, wählen Sie im Saleslogix-Adressbuch Kontakte aus, und klicken Sie dann auf "Mit SLX senden", um den Nachrichtentext und etwaige Anlagen im dazugehörigen Datensatz aufzuzeichnen. Weitere Informationen finden Sie im Thema "Grundlegendes zur E-Mail-Integration".
  • Wählen Sie eine E-Mail-Nachricht in Outlook aus, und klicken Sie dann auf "In Historie aufzeichnen", um den Nachrichtentext und etwaige Anlagen im dazugehörigen Datensatz aufzuzeichnen.
  • Senden der Informationen zu einem Saleslogix-Kontakt im vCard-Dateiformat per E-Mail.
  • Fügen Sie Saleslogix-Kontakte zu Outlook hinzu, fügen Sie Outlook-Kontakte zu Saleslogix hinzu, und synchronisieren Sie Änderungen, die in vorgenommen wurden, mit SaleslogixOutlook
  • Sie können E-Mail-Nachrichten aus Outlook per Ziehen und Ablegen auf die Registerkarte Saleslogix-Historie verschieben.
  • Klicken Sie in der Kontakt- oder Interessentendetailansicht auf die Schaltfläche "E-Mail", um eine neue E-Mail-Nachricht zu öffnen, die an diesen Kontakt oder Interessenten adressiert ist. Weitere Informationen finden Sie unter Siehe Senden von E-Mails an Kontakte oder Interessenten.
  • Klicken Sie in der Ticket-Detailansicht auf die Schaltfläche "E-Mail", um Ticket-Informationen in eine neue E-Mail-Nachricht zu kopieren. Klicken Sie danach in Microsoft Outlook auf "Mit SLX senden", um die Nachricht zu senden und sie in Saleslogix zu speichern. Weitere Informationen finden Sie unter Siehe Senden einer E-Mail-Nachricht aus einem Ticket.
