Auswählen eines Bedingungswertes

Wenn Sie beim Festlegen einer Bedingung im Abfrage-Assistenten das Dialogfeld "Bedingung zuweisen" ausfüllen, wird die Schaltfläche "Durchsuchen" neben dem Feld "Wert" angezeigt. Wenn Sie auf "Durchsuchen" klicken, werden im Dialogfeld "Werte auswählen" alle in der Datenbank befindlichen Werte für das ausgewählte Feld angezeigt. Wählen Sie einen Wert aus, und klicken Sie dann auf "OK".
