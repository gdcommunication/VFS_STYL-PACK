Grundlegendes zur E-Mail-Integration

Dank der E-Mail-Integration sind Sie in der Lage, E-Mail-Nachrichten aus Saleslogix zu senden, wobei ein externes E-Mail-System verwendet wird, oder E-Mails direkt aus einem externen E-Mail-System an einen oder mehrere SaleslogixKontakte oder Interessenten zu senden und diese als SaleslogixHistorieneinträge aufzuzeichnen.

E-Mail-Integration

  • Die Schaltfläche "Mit SLX senden" fügt einen Historiendatensatz in SalesLogix-Client hinzu.
  • Speichern von E-Mail-Anlagen in Saleslogix.
  • Zeichnet in Ihre Saleslogix-Historie aus Ihrem Posteingang oder Ihren gesendeten Objekten auf.
  • Ziehen Sie E-Mail-Nachrichten zur Registerkarte "Saleslogix-Historie" und legen Sie sie dort ab.
  • Verwenden des Saleslogix-Adressbuchs.
  • Hängen Sie Saleslogix-Bibliotheksdokumente an E-Mail-Nachrichten an.
  • Hängen Sie eine Saleslogix-Kontakt-vCard in E-Mail-Nachrichten an.
  • Sie können auch Informationen zu einem Saleslogix-Kontakt von Outlook aus im vCard-Format per E-Mail senden.
  • Schließen Sie Anlagen aus der Registerkarte "Anlagen" ein, indem Sie sie in die Nachricht ziehen.

Wichtige Informationen zur E-Mail-Integration

Wenn Sie die E-Mail-Integrationsfunktionen verwenden, sollten Sie Folgendes beachten:

  • Eingeben von Kontaktnamen: Wenn Sie einen Kontaktnamen ganz oder teilweise in das Feld "An" eingeben und mehrere Kontakte gefunden werden, wird die E-Mail-Adresse des zuerst gefundenen Kontakts verwendet. Wenn Ihr Unternehmen als Datenbankplattform Oracle verwendet, müssen Sie darüber hinaus die Groß- und Kleinschreibung genauestens beachten, damit der Name des Kontakts gefunden werden kann.
  • Aufzeichnen von Notizen/Historien-Daten aus einer von Ihnen gesendeten Nachricht: In jeder E-Mail-Nachricht, die Sie in Ihrem E-Mail-System verfassen, wird die Schaltfläche "Mit SLX senden" angezeigt. Klicken Sie auf diese Schaltfläche, um Ihre Nachricht zu senden und Notiz-/Historiendaten aufzuzeichnen. Die Nachricht wird für jeden Saleslogix-Kontakt oder -Interessenten und die dazugehörige Firma auf der Registerkarte "Notizen/Historie" aufgezeichnet. Saleslogix ermittelt den Kontakt oder Interessenten anhand einer übereinstimmenden E-Mail-Adresse. Wenn Sie beispielsweise in Outlook eine Nachricht an jschmidt@schmidtgmbh.de verfassen und auf "Mit SLX senden" klicken, sucht Saleslogix nach einem Kontakt oder Interessenten mit derselben E-Mail-Adresse. Wenn ein solcher gefunden wird, dann wird die Nachricht in den Notiz-/Historiendaten aufgezeichnet. Wenn keine übereinstimmende E-Mail-Adresse gefunden wird, dann wird in einem Dialog gemeldet, dass der Kontakt nicht gefunden wurde, und es werden keine Notiz-/Historiendaten aufgezeichnet. (Sie können die Notiz-/Historiendaten jedoch aufzeichnen, indem Sie die Nachricht aus Ihrem E-Mail-System zum Saleslogix Client ziehen.) Wenn mehrere Kontakte gefunden werden, werden Sie aufgefordert, einen Kontakt auszuwählen, der die Notiz-/Historiendaten erhalten soll.
    Wenn die Nachricht eine oder mehrere Anlagen hat, werden Sie gefragt, ob eine Kopie der Anlagen in Saleslogix beibehalten werden soll. Wenn Sie auf "Ja" klicken, wird für jeden Kontakt oder Interessenten ein Hyperlink zu dieser Anlage zur Registerkarte "Anlagen" hinzugefügt.
  • Aufzeichnen von Notizen/Historiendaten aus einer von Ihnen empfangenen Nachricht: Im Hauptfenster von Outlook wird die Schaltfläche "In Historie aufzeichnen" angezeigt. Wählen Sie eine Nachricht aus, und klicken Sie auf "In Historie aufzeichnen" (im Menü Extras), oder ziehen Sie die Nachricht aus Outlook in den Saleslogix-Kontaktdatensatz. Anlagen werden wie oben im Zusammenhang mit der Schaltfläche "Mit SLX senden" beschrieben aufgezeichnet.
  • Ausschließen von Kontakten aus Ihrer Organisation: Ihr Systemadministrator kann Regeln für einen Domänennamenausschluss einrichten, um für einige Kontakte (z. B. für Mitarbeiter Ihres Unternehmens) die Aufzeichnung von Notizen/Historiendaten zu verhindern. Dieselben Regeln gelten auch, wenn Nachrichten automatisch für Folgeaktivitäten markiert werden. Weitere Informationen hierzu erhalten Sie bei Ihrem Systemadministrator oder im Thema "Hinzufügen eines Domänenausschlusses" in der Administratorhilfe.
  • Senden von Nachrichten ohne Verbindung zum Server mit Exchange: Das Saleslogix-Adressbuch in Outlook ist nicht von einer Verbindung zum Host-Server mit Exchange, sondern von der Verbindung zur Datenbank (Netzwerk oder remote) abhängig. Wenn Sie auf die Schaltfläche "Mit SLX senden" klicken, wird die Nachricht in Saleslogix in den Notiz-/Historiendaten aufgezeichnet, verbleibt jedoch in Ihrem Outlook-Postausgang, bis eine Verbindung zum Server mit Exchange hergestellt wird.
  • Senden von Nachrichten bei geschlossenem Saleslogix: Wenn der Saleslogix Windows Client beim Klicken auf die Schaltfläche "SLX senden" oder "In Historie aufzeichnen" geschlossen ist, wird eine temporäre Datei in ein temporäres Verzeichnis geschrieben. Wenn Sie SalesLogix-Client öffnen, werden die Notizen-/Historien-Informationen aufgezeichnet.
