Verwenden von Integrationen

Integrationen ermöglichen die Kommunikation zwischen Saleslogix und externen Systemen und Services. Jede Integration übermittelt evtl. Informationen auf unterschiedliche Weise oder weist ein anderes Kommunikationsverhalten auf. Weitere Informationen Beispiel: Die Buchhaltungsintegration mit X3 synchronisiert bestimmte Buchhaltungsinformationen zwischen Saleslogix und X3. Andere Integrationen haben möglicherweise eine einseitige Kommunikation. Beispiel: Eine Integration mit einem Social-Media-Service kann Saleslogix so konfigurieren, dass Informationen vom Service abgerufen werden, um sie anzuzeigen.

Nachdem ein Administrator eine Integration implementiert hat, kommuniziert Saleslogix automatisch mit dem integrierten System oder Service. In einigen Fällen erfordern Integrationen außerdem eine Service-Genehmigung durch Benutzer, bevor die Kommunikation startet.

Integrationen:

  • Buchhaltungsintegrationen

  • Outlook-Synchronisierung

    • Kalenderintegration
