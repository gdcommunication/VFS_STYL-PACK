Verwenden der Konfigurationsansicht der SData-Synchronisierung

Verwenden Sie die SData-Sync-Konfigurationsansicht, um den Saleslogix-Feed einzurichten, der für die Synchronisierung von Saleslogix mit integrierten Systemen verwendet wird.

So konfigurieren Sie SData

  1. Klicken Sie in der Navigationsleiste auf SData-Sync-Konfiguration
  2. Einrichten des Saleslogix Feeds. Verwenden Sie die folgenden Felder, um den Saleslogix-Feed zu konfigurieren. Der Feed wird während der Dateneingabe automatisch aktualisiert.
    1. Wenn Ihr SData-URL "Secure Socket Layer" verwendet, markieren Sie das Kontrollkästchen SSL (Secure Socket Layer) verwenden.
    2. Geben Sie im Feld Server den Namen des Servers für Ihren SData-URL ein.
    3. Geben Sie im Feld Port die Portnummer für Ihren SData-URL ein.
  3. Lassen Sie "ADMIN" im Feld Benutzername unverändert, oder geben Sie den Benutzernamen für die Saleslogix-Datenbank ein.

    Für den Benutzer mit Administratorrechten muss die Windows-Authentifizierung konfiguriert werden. Einzelheiten dazu finden Sie unter dem Thema "Verwenden der Windows-Authentifizierung" in der Administrator-Hilfe.

  4. Geben Sie in das Feld Passwort das Passwort für den Benutzernamen ein.
  5. Lassen Sie den Eintrag " Saleslogix" im Feld Anzeigename unverändert, oder geben Sie den Anzeigenamen an, der für den SData-Feed angezeigt werden soll.
  6. Klicken Sie auf die Schaltfläche Link testen, und zeigen Sie die Informationen im Feld Status an.
    Wenn der Feed nicht erfolgreich getestet wurde, prüfen Sie die Informationsfelder, nehmen Sie die erforderlichen Änderungen vor, und testen Sie den Link erneut.
  7. Klicken Sie auf Speichern.

Detailansicht-Registerkarten

Der untere Abschnitt enthält die Registerkarten der SData Sync-Konfigurationsansicht. Klicken Sie auf einen Hyperlink, um weitere Informationen zu jeder Registerkarte anzuzeigen.
