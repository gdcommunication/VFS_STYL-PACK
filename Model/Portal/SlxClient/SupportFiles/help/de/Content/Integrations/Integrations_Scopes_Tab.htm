Registerkarte "Umfänge" für Integrationen

Verwenden Sie die Registerkarte "Umfänge", um die Umfänge anzuzeigen, die für eine Integration definiert sind. Umfänge bestimmen, welche Informationen aus der Integration für Saleslogix zur Verfügung stehen.

Die Registerkarte "Umfänge" ist nur für Integrationen mit dem Authentifizierungstyp OAuth1 oder OAuth2 verfügbar.

So öffnen Sie Einträge

  1. Öffnen Sie die Integrationsdetailansicht.
  2. Klicken Sie auf die Registerkarte Umfänge.
