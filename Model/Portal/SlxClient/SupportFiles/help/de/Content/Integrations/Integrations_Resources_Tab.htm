Registerkarte "Ressourcen" in Integrationen

Verwenden Sie die Registerkarte "Ressourcen", um integrationsspezifische Ressourcendefinitionen und -verhaltensweisen anzuzeigen. Ressourcen definieren, welche Informationen zwischen Saleslogix und dem Integrationsvertrag zugewiesen werden. Auf dieser Registerkarte können Sie sehen, ob eine Ressource für die Synchronisierung mit der Integration aktiviert ist und in welcher Richtung die Informationen synchronisiert werden.

Die Registerkarte "Ressourcen" ist nur für Integrationen mit einer Synchronisierungsmethode von Client-seitigem Sync oder Server Sync verfügbar.

Vorgehensweise

Öffnen der Registerkarte "Ressourcen"
  1. Öffnen Sie die Integrationsdetailansicht.
  2. Klicken Sie auf die Registerkarte Ressourcen.
Die Felder "Vergleichen"
  • Klicken Sie auf der Registerkarte Ressourcen auf die Schaltfläche Schema verwalten.
    Das Dialogfeld Schema verwalten wird geöffnet.

Diese Ansicht enthält folgende Spalten:

  • Ressource – Name der Ressource im Integrationsvertrag.
  • Saleslogix – Name der zugehörigen Ressource in Saleslogix
  • Aktiviert – Markiert, wenn die Synchronisierung für die Ressource aktiviert ist.
  • Sync-Richtung – Richtung, in der die Informationen synchronisiert werden: in beide Richtungen, NurZuSlx oder NurVonSlx.
