Grundlegendes zur Kontaktintegration

Wenn Sie Kontakte zwischen Saleslogix und einer unterstützten Integration synchronisieren, kann ein Kontaktdatensatz in einer der Anwendungen geändert und die Änderung in beiden Anwendungen reflektiert werden. Ihr Administrator muss eine Berechtigung konfigurieren und gewähren, bevor Sie Kontakte mit einer Integration synchronisieren können.

During a synchronization cycle, Saleslogix compares fields in both systems, and makes changes to ensure both systems contain the same information. Only data in mapped fields between the applications is transferred. The data is compared and changes are made to ensure the information matches. For example, if a record has been added to one system and not to the other, that record is added to the incomplete system.

Kontakt-Synchronisierungsverhalten

  • Kontakte werden anhand der E-Mail-Adressen verknüpft Spezifisches Übereinstimmungsverhalten finden Sie unter dem Thema "Übereinstimmen von Datensätzen zwischen Saleslogix und Integrationen".
  • Wenn derselbe Datensatz während desselben Synchronisierungszyklus aktualisiert wird, gewinnt die letzte Änderung.
  • Durch Löschen eines Kontaktes in Saleslogix wird der Kontakt aus dem Ordner Kontaktsynchronisierung zu einem Ordner mit gelöschten Elementen verschoben.
  • Durch Löschen eines Kontakts in einer integrierten Anwendung wird der Kontakt aus der Kontaktsynchronisierungsgruppe entfernt, der Kontakt wird jedoch nicht aus Saleslogix gelöscht.

Spezifische Kontaktintegrationsszenarios finden Sie unter Kontaktsynchronisierungsszenarien. 

Synchronization is restricted by the built-in data security associated with the user who is logged on to Saleslogix. The amount of data available to transfer is limited by the security rights of the user logged in.

 

Es empfiehlt sich nicht, den Remote Client gleichzeitig mit der Datenbank der Zentrale und mit Microsoft Outlook zu synchronisieren. Dies reduziert die Synchronisierungsgeschwindigkeit deutlich und verursacht Speicherprobleme. Saleslogix empfiehlt empfiehlt, die Synchronisierung mit Outlook vor oder nach der Synchronisierung mit der Remote-Datenbank durchzuführen.

Vorgehensweise

  • Erstellen der Kontaktsynchronisierungsgruppe
  • Hinzufügen von Kontakten zur Kontaktsynchronisierungsgruppe
  • Konfigurieren der Kontaktsynchronisierungsgruppe
  • Festlegen von Kontaktoptionen
