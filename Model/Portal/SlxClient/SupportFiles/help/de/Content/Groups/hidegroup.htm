Ausblenden von Gruppen

Sie können eine Gruppe ausblenden, die Sie nicht mehr anzeigen, aber auch nicht löschen möchten.

So blenden Sie eine Gruppe aus

  1. Klicken Sie in einer Listenansicht mit der rechten Maustaste auf eine Gruppenregisterkarte.
  2. Klicken Sie auf Ausblenden.

Die Standardgruppe kann nicht ausgeblendet werden. Wenn die Gruppe, die Sie ausblenden möchten, die Standardgruppe ist, müssen Sie zunächst in den Optionen eine neue Standardgruppe auswählen.
