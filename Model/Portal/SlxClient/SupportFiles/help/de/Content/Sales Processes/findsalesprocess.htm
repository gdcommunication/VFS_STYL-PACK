Suchen von Verkaufsprozessen

Verkaufsprozesse erleichtern die Verwaltung der Verkaufs-Pipeline. Sie können einen Verkaufsprozess an eine geöffnete Verkaufschance anhängen, indem Sie einen Prozess in einer Liste verfügbarer Prozesse auswählen.

So suchen Sie Verkaufsprozesse

  1. Öffnen Sie die Ansicht Verkaufschancendetail.
  2. Klicken Sie auf die Registerkarte Verkaufsprozess.
  3. Klicken Sie auf den Dropdown-Pfeil zu Verkaufsprozess, um eine Liste der verfügbaren Verkaufsprozesse anzuzeigen.
