Arbeiten mit Cockpits

Cockpits werden auf der Begrüßungsseite als Registerkarten angezeigt und enthalten anpassbare, vordefinierte Analysen in Form von Diagramm- und Listen-Widgets. Die Begrüßungsseite kann mehrere Instanzen eines Cockpits enthalten, jedes Cockpit kann mehrere Widgets enthalten.

Es gibt zwei vordefinierte Cockpit-Registerkarten: "Mein Cockpit" und "Vertrieb". Jede Registerkarte verfügt über mehrere vordefinierte Widgets. Sie können Widgets hinzufügen und ändern, um andere Daten bzw. dieselben Daten auf andere Weise anzuzeigen.

Im Kontextmenü auf einer Cockpit-Registerkarte können Sie eine neue Registerkarte hinzufügen, Registerkarten ein- oder ausblenden und löschen oder Registerkarten für Ihr Team oder Ihre Abteilung freigeben. Per Ziehen und Ablegen können Sie die Widgets auf einer Registerkarte neu anordnen. Die Größe von Widgets passt sich automatisch an den Daten- und Diagrammtyp sowie an die Spaltengröße an.

Sie können auch über den Hyperlink "In Cockpit hochstufen" alle Daten für eine Gruppe aus einer Listenansicht in ein Gruppenlisten-Widget im Cockpit übertragen.

Wenn Sie nicht der Zugriffsberechtigte für das Cockpit sind, müssen Sie eine Kopie erstellen, die Sie freigeben und ändern können.

Vorgehensweise

Zugreifen auf ein Cockpit
  • Klicken Sie auf der Navigationsleiste auf Willkommen.
Hinzufügen einer neuen Registerkarte
  1. Klicken Sie mit der rechten Maustaste auf eine beliebige Cockpit-Registerkarte, und klicken Sie auf Neue Registerkarte.
  2. Geben Sie in das Feld Titel einen Namen für die Registerkarte ein.
  3. Wählen Sie bei Bedarf die Vorlage aus, die den Layout-Typ beschreibt, den Sie für Ihre Registerkarte verwenden möchten.
  4. Wenn Sie die ausgewählte Vorlage als Standardlayout festlegen möchten, wählen Sie Als Standard festlegen aus.
  5. Klicken Sie auf "OK".
Kopieren einer Registerkarte
  1. Klicken Sie mit der rechten Maustaste auf die Cockpit-Registerkarte, und klicken Sie auf Registerkarte kopieren.
  2. Geben Sie in das Feld Titel einen Namen für die Registerkarte ein.
  3. Klicken Sie auf OK.
    Die Registerkarte und ihr Inhalt werden auf eine neue Registerkarte kopiert.
Löschen einer Registerkarte
  1. Klicken Sie mit der rechten Maustaste auf die Cockpit-Registerkarte, und klicken Sie auf Registerkarte löschen.
  2. Klicken Sie auf Ja, um den Vorgang zu bestätigen.
    Das Cockpit und sein gesamter Inhalt werden aus dem Arbeitsbereich entfernt.
Ausblenden einer Registerkarte
  • Klicken Sie mit der rechten Maustaste auf die Cockpit-Registerkarte, und klicken Sie auf Registerkarte ausblenden.
    Die Registerkarte wird ausgeblendet.
Einblenden einer Registerkarte
  1. Klicken Sie mit der rechten Maustaste auf eine Cockpit-Registerkarte, und klicken Sie auf Registerkarte anzeigen.
  2. Klicken Sie neben dem Namen der Registerkarte, die Sie anzeigen möchten, auf Anzeigen.
    Die Registerkarte wird im Cockpit angezeigt.
Freigeben einer Registerkarte für andere Benutzer
  • Klicken Sie mit der rechten Maustaste auf die Cockpit-Registerkarte, und klicken Sie auf Registerkarte freigeben.

Führen Sie einen der folgenden Schritte aus:

  • Klicken Sie auf Alle, um die Registerkarte für alle Benutzer freizugeben.
  • Klicken Sie auf die Schaltfläche Hinzufügen, suchen Sie die Benutzer, Teams oder Abteilungen, für die Sie die Registerkarte freigeben möchten, und klicken Sie auf OK.
    Falls Sie fälschlicherweise einen Benutzer, ein Team oder eine Abteilung hinzugefügt haben, wählen Sie die Einträge aus, die aus der Liste entfernt werden sollen, und klicken auf Entfernen.

Wenn Sie eine Registerkarte "Cockpit" freigeben, die eine Gruppe enthält, muss die Gruppe ebenfalls für diese Benutzer freigegeben werden, damit die Gruppe auf der Registerkarte "Cockpit" angezeigt wird.
Ändern der Registerkartenbeschriftung oder der Spaltengröße oder Einrichten einer Registerkarte als Standard
  1. Klicken Sie mit der rechten Maustaste auf die Cockpit-Registerkarte, und klicken Sie auf Optionen bearbeiten.
  2. Geben Sie einen neuen Titel ein, um die Beschriftung der Registerkarte zu ändern.
  3. Wählen Sie das für diese Registerkarte gewünschte Spaltenlayout:
    • Eine Spalte
    • Unterteilung in zwei Spalten
    • Zweispaltig, links größer
    • Zweispaltig, rechts größer
  4. Wählen Sie die Option Als Standard festlegen aus, um die aktuelle Registerkarte als Standard festzulegen.
    Die Standardregisterkarte kann weder ausgeblendet noch gelöscht werden.
  5. Klicken Sie auf "OK".
