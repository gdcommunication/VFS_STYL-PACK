Support technique

Les clients Saleslogix ayant souscrit à un contrat d’assistance ont accès au support technique directement auprès de Swiftpage et de ses partenaires commerciaux agréés. Nos services sont joignables par téléphone aux heures de bureau du lundi au vendredi, hors jours fériés. Les informations de contact sont accessibles sur le web à l'adresse http://www.saleslogix.com.

  • Les partenaires disposant d'un contrat de support technique valide et d'un code d'accès Internet peuvent adresser leurs demandes d'assistance par voie électronique sur le portail Web d'assistance partenaire Saleslogix : http://www.saleslogix.com/partners.
  • Les clients disposant d'un contrat de support technique valide et d'un code d'accès Internet peuvent adresser leurs demandes d'assistance par voie électronique sur le portail Web d'assistance client Saleslogix : http://www.saleslogix.com/customers.
